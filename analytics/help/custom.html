Rapports personnalisés

Créez des rapports d'analyse personnalisés qui présentent des données et des mesures sélectionnées pendant un certain temps.

Le fuseau horaire du rapport

Les données des rapports personnalisés sont basées sur le fuseau horaire des rapports du compte ; pour configurer votre fuseau horaire:

  1. Dans Beacon Studio, développez le menu déroulant Admin.
  2. Sélectionnez Paramètres du compte.
  3. Définissez votre fuseau horaire actuel.
  4. Cliquez sur Enregistrer.
La modification du fuseau horaire des rapports n'affecte que les données à venir, et les modifications ne sont pas appliquées rétroactivement.

Données du rapport
  • Inclure les comptes - Incluez les données du compte Video Cloud actuel ou sélectionnez plusieurs comptes
  • Type de rapport
    • Une fois - Le rapport est exécuté une fois, immédiatement
    • Plage de dates: pour les rapports ponctuels, spécifiez la période pour les données d'analyse
    • Récurrent - Le rapport peut être programmé pour s'exécuter sur une base quotidienne, hebdomadaire ou mensuelle
    • Fréquence - Pour les rapports récurrents
      • Quotidien - Le rapport couvre une journée entière
      • Hebdomadaire - Le rapport couvre une semaine entière, définie du lundi au dimanche
      • Mensuel - Les rapports couvrent un mois entier
      • Les rapports récurrents sont exécutés après le rapprochement des analyses, généralement 3 à 5 jours après la fin d'une période de reporting.
  • Groupes de données: les groupes de données déterminent les champs de données inclus dans le rapport
    • Les champs de données qui apparaissent sont basés sur les groupes de données que vous sélectionnez
    • Vous pouvez sélectionner plusieurs groupes de données, mais certaines combinaisons ne sont pas valides.
    • Au moins un groupe de données doit être sélectionné
  • Champs de données: lorsque vous sélectionnez un groupe de données, les champs de données qui peuvent être renvoyés sont sélectionnables
    • Les champs grisés ne peuvent pas être renvoyés pour le groupe de données sélectionné
  • Cliquez sur Enregistrer pour enregistrer vos modifications

Ajouter des filtres de rapport

Des filtres peuvent être ajoutés pour limiter la quantité de données renvoyées.

  • Lors de l'utilisation de plusieurs filtres, une opération AND sera utilisée
  • Un filtre ne s'affichera dans votre rapport que si vous avez sélectionné son groupe de données correspondant.
    • Par exemple, si vous souhaitez appliquer un filtre sur la balise vidéo, vous devez sélectionner la dimension Vidéo

Vous pouvez appliquer un filtre sur les éléments suivants:

  • Vidéo: un ou plusieurs identifiants vidéo actifs dans le compte
  • Video Tag - Une ou plusieurs balises vidéo
  • Champs personnalisés: un ou plusieurs champs personnalisés du compte
  • Joueur: saisissez un ou plusieurs identifiants de lecteur s, séparés par une virgule
  • Type de source
  • Type d'appareil
  • OS de l'appareil
  • Fabricant de l'appareil
  • Type de navigateur
  • Termes de recherche
  • Pays
  • Domaine de destination

Aperçu du rapport

Les données affichées dans l'aperçu du rapport sont des exemples de données, et non des données réelles de votre compte Video Cloud.

  • La section Aperçu du rapport affiche les colonnes de données sélectionnées, remplies avec des exemples de données
  • Cliquez pour supprimer l'élément de données associé
  • L'ordre de tri de l'aperçu n'a aucun effet sur l'ordre de tri du rapport téléchargé
  • Vous pouvez utiliser un tableur pour trier les données téléchargées

Configuration et livraison du rapport
  • Report Name: commencez par donner un nom significatif au rapport
  • Format
    • CSV - Valeurs séparées par des virgules
    • XLSX - Microsoft Excel
  • Envoyez-moi un e-mail lorsque mon rapport est prêt - Recevez une notification par e-mail lorsque le rapport est terminé
  • Notifier les autres - Saisissez les adresses e-mail séparées par une virgule
  • Dans l'e-mail de Brighcove, le lien pour télécharger le rapport est valide pendant 90 jours
  • Pour les rapports ponctuels,
    • Cliquez sur Demande de rapport
    • Le rapport sera téléchargé lorsqu'il sera terminé
  • Pour les rapports récurrents,
    • Cliquez sur Schedule Report
    • Le rapport sera mis en file d'attente et exécuté lorsque les données seront prêtes
  • Pour voir tous les rapports planifiés, cliquez sur Rapports planifiés dans le volet de navigation de gauche
