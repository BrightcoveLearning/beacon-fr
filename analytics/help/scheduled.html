Rapports planifiés

Gérez vos rapports d'analyse personnalisés que vous avez définis pour être exécutés de manière récurrente.

Les comptes d'entreprise peuvent avoir jusqu'à 20 rapports planifiés. Les comptes professionnels sont limités à 5 rapports.
Liste des rapports
  • Nom: nom du rapport planifié
  • Frequency: intervalle de temps pendant lequel le rapport sera exécuté
  • État du dernier rapport: si le dernier rapport a été livré ou non
  • Destinataires du rapport: liste des adresses e-mail qui recevront le rapport
  • Actif ou inactif: définissez le statut du rapport
    • Actif - Le traitement du rapport est planifié
    • Inactif - Le rapport ne sera pas traité
Détails du rapport
  1. Pour développer les détails du rapport, cliquez sur + en regard du nom du rapport.
  • Date de création: date à laquelle le rapport a été exécuté
  • Comptes inclus: liste des comptes à partir desquels les données ont été collectées
  • Groupes de données - Groupes de données inclus
  • Filtres: filtres utilisés pour limiter les résultats
  • Données de groupe - Champs de groupe et de données inclus
  • Données de performance: statistiques de performance des vidéos et des lecteurs incluses
  • Données d'engagement: données qui montrent comment les utilisateurs interagissent avec vos vidéos
  • Données d'utilisation -
  • Champs personnalisés - Champs personnalisés inclus
  • Trié par: colonnes de données triées
  • Notifications par e-mail à: liste des adresses e-mail qui reçoivent le rapport
  • Prochain rapport programmé: date à laquelle le prochain rapport sera exécuté
Modifier le rapport
  1. Pour développer les détails du rapport, cliquez sur + en regard du nom du rapport.
  2. Cliquez sur Modifier.
  3. Apportez des modifications à la définition du rapport.
  4. Cliquez sur Planifier le rapport.
Si le rapport inclut plusieurs comptes Video Cloud, vous devez avoir accès à tous les comptes pour pouvoir le modifier.
Exécuter le rapport
  1. Pour développer les détails du rapport, cliquez sur + en regard du nom du rapport.
  2. Cliquez sur Exécuter le rapport.
Les données actuellement disponibles pour le prochain rapport programmé seront téléchargées.
Rapport en double
  1. Pour développer les détails du rapport, cliquez sur + en regard du nom du rapport.
  2. Cliquez sur Dupliquer
  3. Apportez des modifications à la définition du rapport.
  4. Mettez à jour le nom du rapport.
  5. Cliquez sur Planifier le rapport.
Le type de rapport peut être modifié de Récurrent à Unique, ce qui vous permet d'exécuter le rapport immédiatement.
Supprimer le rapport
  1. Pour développer les détails du rapport, cliquez sur + en regard du nom du rapport.
  2. Cliquez sur Supprimer.
  3. Pour confirmer la suppression, cliquez sur Supprimer le rapport.
Si le rapport inclut plusieurs comptes Video Cloud, vous devez avoir accès à tous les comptes pour pouvoir le supprimer.
