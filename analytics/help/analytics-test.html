Analyses

Obtenez rapidement des données d'analyse en temps réel pour vos vidéos et vos lecteurs.

Performance

Le rapport de performance présente des mesures, notamment les vues de vidéos, les sources de trafic, les types d'appareils, les systèmes d'exploitation des appareils, etc.

  • Liste déroulante Vues vidéo - Sélectionnez les mesures du rapport:
    • Video Views: nombre de fois qu'une vidéo commence à être lue
    • Impressions vidéo: nombre de fois où la vidéo a été chargée dans un lecteur et prête à être utilisée
    • Taux de lecteur: nombre de vues de la vidéo divisé par le nombre
    • Viewed Minutes: nombre total de minutes pendant lesquelles la vidéo a été visionnée
    • Score d'engagement: pourcentage moyen de visionnages par vidéo visionnée
    • Visionneuses uniques quotidiennes: nombre moyen de spectateurs uniques quotidiens pour la période sélectionnée
    • Bande passante vidéo: bande passante pour les vidéos uniquement ; peut être retardée de 3 à 5 jours
    • Bande passante totale: inclut la bande passante pour les vidéos, les images et le chargement des lecteurs ; peut être retardée de 3 à 5 jours
  • Liste déroulante Performances - Basculez entre les rapports Performance et Engagement
  • Liste déroulante des appareils
    Le graphique montre le nombre de visionnages de vidéos par jour pendant la période sélectionnée.

    Le corps du rapport affiche les éléments suivants, lorsque chaque dimension est sélectionnée:
    • Vidéo: nom de la vidéo, identifiant de la vidéo, durée et détails sur les performances de chaque vidéo
    • Player: détails sur les performances de chaque lecteur
    • Source of Traffic: données de performance pour chaque source de trafic
      • Parrainage - L'utilisateur a suivi un lien vers la vidéo
      • Tablette - Comprend les iPads, les tablettes Android, etc.
      • Mobile - Téléphones portables
      • Téléviseur connecté/console de jeux/appareil compagnon - Comprend les téléviseurs intelligents, les décodeurs et les consoles de jeux
      • Autre - Les agents utilisateurs ne sont mappés à aucun des éléments ci-dessus
    • Display Domain: données de performance pour le domaine sur lequel se trouvait l'utilisateur lorsqu'il a regardé une vidéo
    • Pays: données de performance pour chaque pays dans lequel les vidéos ont été visionnées
    • Device: données de performance pour chaque combinaison Device Type/Device Operating System utilisée pour visionner une vidéo
    • Device Type: détails des performances par type d'appareil
      • Ordinateur de bureau - Comprend les ordinateurs de bureau et portables
      • Direct - L'utilisateur a tapé le lien directement ou le référent n'a pas été déterminé
      • Recherche sécurisée: l'utilisateur s'est connecté avec un compte Google et a effectué une recherche
      • Recherche organique: l'utilisateur a effectué une recherche et a cliqué sur un résultat
      • Recherche payante - L'utilisateur a effectué une recherche et a cliqué sur un résultat payant
    • Device OS: données de performance pour chaque système d'exploitation de l'appareil
    • Device Manufacturer: données de performance pour chaque fabricant d'appareils
    • Browser Type: données de performance pour chaque type de navigateur
    • Day - Données de performance pour chaque jour

Engagement

Le rapport d'engagement montre comment les utilisateurs interagissent avec vos vidéos (le pourcentage d'une vidéo que les spectateurs regardent).

  • Liste déroulante Engagement - Basculez entre les rapports Performance et Engagement
  • Liste déroulante des lecteur s
    Le graphique montre l'engagement vidéo global pour la période sélectionnée.

    Le corps du rapport affiche les éléments suivants, lorsque chaque dimension est sélectionnée:
    • Vidéo: nom de la vidéo, identifiant de la vidéo, durée et détails d'engagement pour chaque vidéo
    • Joueur: détails sur l'engagement de chaque lecteur
    • Day - Détails de l'engagement pour chaque jour
  • Calculer l'engagement vidéo
    • L'engagement est une mesure des points de vue et non des spectateurs
    • La vidéo est divisée en 100 parties égales et les vues sont comptées dans chaque partie
    • Un spectateur qui regarde une partie 5 fois n'est pas différent de cinq spectateurs qui la regardent une fois.
  • Exemples d'engagement vidéo
    • Pas encore de montre - Engagement Score: 0
    • Un spectateur charge la page et regarde l'intégralité de la vidéo - Engagement Score: 100 (100 pour cent vues/1 vue)
    • Le même spectateur revient en arrière et le regarde à nouveau (navigateur non rechargé) - Score d'engagement: 200 (200 % vues/1 vue)
    • Un autre spectateur charge la page et regarde l'intégralité de la vidéo - Engagement Score: 150 (300 pour cent vues/2 vues)
  • Données récentes et historiques
    • Pendant 32 jours, Brightcove conserve des données d'analyse vidéo détaillées (vues pour chaque centième de la durée de la vidéo).
    • Après 32 jours, les données d'analyse sont déplacées vers les données historiques. Seules les vues aux points de 1 %, 25 %, 50 %, 75 % et 100 % sont conservées.
    • Les rapports exportés incluent uniquement les vues aux points de 1 %, 25 %, 50 %, 75 % et 100 %, même au cours des 32 derniers jours.
    • Utilisez l'API Analytics pour obtenir des données détaillées sur l'engagement au cours des 32 derniers jours.

Ajouter un filtre

Utilisez des filtres pour limiter les données qui apparaissent dans votre rapport.

  1. Cliquez sur la pilule + Ajouter des filtres.
  2. Les données Analytics peuvent être filtrées par balises, champs personnalisés, pays et types d'appareils.
    Configurez les critères de filtre à l'aide du panneau de filtres
  3. Cliquez sur Appliquer les filtres.

Exporter les données du

Les données du rapport peuvent être téléchargées sous forme de fichier CSV ou au format Excel.

L'exportation enregistre toutes les données directement à partir d'un appel d'API Analytics, sans filtrage. Cela vous permet de manipuler les données comme vous le souhaitez.
  1. À droite de Exporter sous, cliquez sur CSV ou XLSX.
  2. Ouvrez ou enregistrez le fichier sur votre ordinateur.
