{{page.title}}

{{page.description}}
  • Passez la souris sur les listes de lecteur, puis cliquez sur Ajouter pour ajouter la liste de lecteur souhaitée à la page.
  • Cliquez sur Supprimer pour supprimer une liste de lecteur de la page.
  • Cliquez sur X pour fermer le panneau de sélection de la playlist.
  • Le champ Rechercher en haut de la page peut être utilisé pour rechercher une liste de lecteur.
