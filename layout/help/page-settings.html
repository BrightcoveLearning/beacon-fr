Dans la boîte de dialogue de mise en page, vous pouvez:

  • Modifiez le titre de la page et les traductions du titre de la page si vous utilisez plusieurs langues.
  • Modifier la disponibilité des pages pour le calendrier, les appareils et/ou l'emplacement. Cliquez sur le bouton correspondant pour modifier les valeurs.
  • Archivez la page. Notez qu'aucune confirmation ne sera proposée, la page est immédiatement archivée.
