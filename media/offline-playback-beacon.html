Présentation

La lecteur hors ligne permet aux utilisateurs de télécharger du contenu vidéo sur leur appareil mobile et de le regarder plus tard lorsqu'ils ne sont pas connectés. Il se peut que certaines vidéos ne soient pas disponibles pour une lecteur hors ligne. Si un bouton de téléchargement apparaît pour le contenu, les utilisateurs peuvent télécharger des épisodes et des films pour les regarder plus tard.

À partir de l'application Brightcove Beacon, vous pouvez accéder à la liste de tous vos contenus téléchargés. Ici, vous pouvez lire et supprimer les vidéos que vous avez téléchargées. Lorsque vous êtes en ligne, vous pouvez renouveler un actif expiré.

Caractéristiques

Utilisateurs

  • Le contenu configuré pour la lecteur hors ligne sera doté d'un bouton de téléchargement
  • Téléchargez des vidéos sur les applications iOS et Android pour appareils mobiles et tablettes
  • Téléchargez des vidéos protégées par DRM et des vidéos claires (non cryptées) et regardez-les lorsque vous n'êtes pas connecté
    • Le contenu sera lu sur l'appareil si le spectateur se déplace vers une autre région qui n'est pas prise en charge par les paramètres de droits jusqu'à ce qu'il se reconnecte au réseau mobile ou Wi-Fi
  • Possibilité d'annuler, de suspendre ou de supprimer un téléchargement
  • Une fois le téléchargement terminé, le réseau peut être déconnecté ou indisponible et les vidéos peuvent être lues à partir de l'appareil
  • L'audio et les sous-titres doivent être disponibles lorsque la vidéo est téléchargée
  • Possibilité de supprimer ou de renouveler un actif lorsqu'il arrive à expiration
  • Si l'abonnement du spectateur devient invalide pour une raison quelconque, la lecteur sera bloquée lorsqu'il se reconnectera au réseau
  • Si l'utilisateur déconnecte son compte sur l'appareil, les vidéos téléchargées seront supprimées de la machine de l'utilisateur

Clients

  • Vous pouvez sélectionner les vidéos individuelles qui peuvent être téléchargées dans Video Cloud, mais elles doivent faire partie d'un abonnement/package dont le téléchargement est activé.
  • Les publicités côté serveur (SSAI) ne sont pas diffusées pour le contenu lu à partir de l'appareil

Exigences

Activez votre compte

Pour la lecteur hors ligne, procédez comme suit:

  • Contactez votre responsable de compte Brightcove pour activer la lecteur hors ligne de votre compte Video Cloud Studio. Notez que la lecteur hors ligne est un module complémentaire payant.
  • L'équipe d'accueil activera votre compte Brightcove Beacon pour la lecteur hors ligne
    • Les applications peuvent être configurées pour offrir la lecteur hors ligne en tant que fonctionnalité premium, disponible via un abonnement

Gérez votre contenu

Assurez-vous que votre contenu est disponible pour une lecteur hors ligne:

  • Dans Video Cloud Studio, ingérez des vidéos à l'aide d'un profil d'acquisition qui produit une ressource téléchargeable (voir les étapes ci-dessous)

    L'ingestion de contenu avec une ressource téléchargeable augmentera les coûts de stockage des clients.
  • Dans Video Cloud Studio, définissez les vidéos pour activer la lecteur hors ligne (voir les étapes ci-dessous)
  • Dans Beacon CMS, définissez les ressources qui seront disponibles pour la lecteur hors ligne (voir les étapes ci-dessous)

Configuration dans Video Cloud Studio

Suivez ces étapes pour activer les ressources vidéo pour la lecteur hors ligne dans Video Cloud Studio:

  1. Lorsque vous mettez en ligne ou retranscodez une vidéo, assurez-vous d'utiliser un profil d'acquisition qui inclut une ressource téléchargeable (MP4).

    L'ingestion de contenu avec une ressource téléchargeable augmentera les coûts de stockage des clients.

    Par exemple, votre profil peut être similaire à celui-ci:

  2. Dans le module Médias, sélectionnez une ressource vidéo que vous souhaitez activer pour la lecteur hors ligne.

  3. Dans la section Informations sur la vidéo, sélectionnez Modifier.

  4. Sélectionnez Activer la lecteur hors ligne. Vous procéderez ainsi pour toutes les ressources téléchargeables.

  5. Cliquez sur Enregistrer.

Configuration dans Beacon CMS

La lecteur hors ligne peut être définie dans Beacon pour répondre à vos objectifs commerciaux.

Abonnements

Vous pouvez proposer la lecteur hors ligne en tant que fonctionnalité premium aux abonnés en activant la lecteur hors ligne dans les paramètres du plan d'abonnement. La lecteur hors ligne ne sera disponible que pour les abonnés de ce forfait.

  1. Dans Brightcove Beacon CMS, cliquez sur Commerce.

  2. Sélectionnez SVOD/AuthVOD Plans et cliquez sur le nom de votre package.

  3. Pour Autoriser les téléchargements ? , sélectionnez Oui.

  4. Cliquez sur Modifier le package.
  5. Pour plus d'informations sur les plans de monétisation, consultez ce qui suit:
    • Création d'un plan SVOD
    • Implémentation d'AuthVOD

Synchronisation et nettoyage du cache

Sync vidéos

Les modifications apportées à vos actifs dans Video Cloud Studio seront automatiquement synchronisées avec Beacon CMS.

Vider le cache

Pour voir immédiatement vos modifications dans votre application Beacon, actualisez le cache de votre application.
