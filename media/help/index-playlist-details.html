Rechercher des vidéos

Pour rechercher une série, une saison, un épisode ou un film, saisissez des termes de recherche et appuyez sur Entrée.

Les champs vidéo suivants seront recherchés:

  • ID de la vidéo
  • Nom
  • Brève description
  • Description longue
  • Balises
  • ID de référence
  • Champs personnalisés

Créer une liste de lecteur

Des listes de lecteur manuelles peuvent être créées pour regrouper une série de vidéos dont vous pouvez contrôler l'ordre de lecteur.

  1. Cliquez sur + Créer une playlist dans le menu de navigation de gauche.
