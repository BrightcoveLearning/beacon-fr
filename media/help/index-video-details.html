Boutons de barre

Supprimer la vidéo
  1. Cliquez sur Supprimer.
Capture de nouvelles images
  1. Utilisez le lecteur de prévisualisation en haut de la page pour lire la vidéo.
  2. Mettez la vidéo en pause lorsque vous arrivez à l'image que vous souhaitez utiliser pour l'affiche/la vignette.
  3. Cliquez sur Capture.
  4. Sélectionnez cette option pour utiliser l'image comme affiche et/ou vignette.
  5. Cliquez sur Enregistrer.

Détails de la vidéo

Les détails de la vidéo sont organisés en onglets, chaque onglet comportant une ou plusieurs sections. Cliquez sur Modifier ou double-cliquez dans une section. Certaines sections permettent de modifier directement les détails. Cliquez sur Enregistrer pour enregistrer les modifications. Les sections sans bouton Enregistrer seront automatiquement enregistrées lorsque vous quitterez le champ via la touche Tabulation.

Onglet Aperçu

Informations sur la vidéo

Contient des informations de base sur une vidéo.

  • Nom - 255 caractères maximum
  • ID: identifiant vidéo attribué par le système (unique sur tous les comptes Brightcove)
  • Brève information - 250 caractères maximum
  • Tags: une ou plusieurs balises, séparées par des virgules. Vous pouvez utiliser des balises pour ajouter des vidéos à des listes de lecteur intelligentes. 128 caractères maximum.
  • ID de référence: identifiant unique attribué par un éditeur ; 150 caractères maximum
  • Projection vidéo: indique comment la vidéo a été enregistrée
  • Activer la lecteur hors ligne: lorsque cette case est cochée, permet d'enregistrer la vidéo localement et de la visionner hors ligne
  • URL du lien associé: URL Web standard commençant par http://ou https://limitée à 255 caractères
  • Related Link Text - Texte du lien associé limité à 255 caractères
  • Description longue - 5 000 caractères maximum
  • Publicité
    • Gratuit - Les publicités ne seront pas diffusées lors de la lecteur de la vidéo
    • Prise en charge de la publicité - Les publicités peuvent être diffusées lors de la lecteur
  • Clés publicitaires: transmises dans l'URL du serveur publicitaire
  • Partage: indique si la vidéo a été partagée sur un autre compte
Détails
Informations sur le type vidéo
  • Le type de vidéo détermine les détails vidéo supplémentaires qui peuvent être configurés
    • Film: indique que la vidéo est un film. Lorsqu'une nouvelle vidéo est téléchargée, le type de vidéo est défini sur Film.
    • Episode - Indique que la vidéo est un épisode d'une saison. Le panneau Informations sur la vidéo affiche le numéro de l'épisode et la saison à laquelle la vidéo est attribuée.
    • Bande-annonce: indique que la vidéo est utilisée comme bande-annonce. Les vidéos ajoutées en tant que bande-annonce d'un film ou d'un épisode verront automatiquement leur type de vidéo remplacé par Bande-annonce.
Casting
  • Utilisé pour ajouter des informations de diffusion pour une vidéo
  • Tous les champs prennent en charge la recherche par saisie anticipée ; au fur et à mesure que vous tapez, les entrées correspondantes précédentes apparaissent et vous pouvez cliquer dessus pour les ajouter à la sélection
Catégorisation

La section Catégorisation permet de configurer:

  • Genres - Genres auxquels la vidéo est associée ; le genre est utilisé pour déterminer quel contenu associé s'affiche pour le contenu non épisodique sur la page de détails et le lecteur
  • Listes de lecteur manuelles: listes de lecteur auxquelles cette vidéo appartient ; pour plus d'informations, voir Création et gestion de listes de lecteur
  • Dossier: dossier auquel la vidéo est attribuée ; voir Organisation des vidéos à l'aide de dossiers pour plus d'informations
Champs personnalisés
  • Utilisé pour afficher et modifier le champ personnalisé de la vidéo
  • Un astérisque (*) en regard du nom du champ indique que le champ est obligatoire
