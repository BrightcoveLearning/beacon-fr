  2. Cliquez sur Publier sur le Web.
  3. Sélectionnez un lecteur .

Rechercher des vidéos

Pour rechercher une série, une saison, un épisode ou un film, saisissez des termes de recherche et appuyez sur Entrée.

Les champs vidéo suivants seront recherchés:

  • ID de la vidéo
  • Nom
  • Brève description
  • Description longue
  • Balises
  • ID de référence
  • Champs personnalisés

Téléchargez des vidéos
  1. Cliquez sur Télécharger des vidéos dans le volet de navigation de gauche.
  2. Choisissez un profil d'ingestion.
  3. Parcourez et sélectionnez les vidéos à mettre en ligne.
