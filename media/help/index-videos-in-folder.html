Vidéos dans un dossier

{{page.description}}

Publier des vidéos
  1. Sélectionnez une vidéo.
  2. Cliquez sur Publier sur le Web.
  3. Sélectionnez un lecteur .
  4. Cliquez sur le lien d'aperçu ou copiez le code intégré.

Modifier les détails d'une vidéo pour plusieurs vidéos
  1. Sélectionnez une ou plusieurs vidéos, puis cliquez sur Édition rapide.
  2. Apportez les modifications appropriées.
  3. Cliquez sur Enregistrer.

Panneau Afficher les détails de la vidéo
  1. Sélectionnez une vidéo, puis cliquez sur Aperçu rapide.

Cliquez à nouveau sur le bouton pour fermer le panneau.

Ajouter des vidéos à un dossier
  1. Sélectionnez une ou plusieurs vidéos.
  2. Cliquez sur Plus > Déplacer vers le dossier.
  3. Sélectionnez un dossier.
  4. Cliquez sur Déplacer.

Ajouter des vidéos à une liste de lecteur manuelle
  1. Sélectionnez une ou plusieurs vidéos.
  2. Cliquez sur Plus > Ajouter à la liste de lecteur.
  3. Cliquez sur le nom d'une liste de lecteur.
  4. Cliquez sur Ajouter.
Détails
Activer/Désactiver une vidéo
  1. Sélectionnez une ou plusieurs vidéos.
  2. Cliquez sur Plus > Activer ou Désactiver.

Partager des vidéos sur un autre compte
  1. Sélectionnez une ou plusieurs vidéos.
  2. Cliquez sur Plus > Partager.
  3. Sélectionnez un compte d'affilié sur lequel partager.
  4. Cliquez sur Partager.
Détails
Supprimer la vidéo du dossier
  1. Sélectionnez une vidéo.
  2. Cliquez sur Supprimer les vidéos > Supprimer les vidéos du dossier.
Supprimer des vidéos
  1. Sélectionnez une vidéo.
  2. Cliquez sur Supprimer les vidéos > Supprimer les vidéos du système.
Afficher les détails des vidéos
  1. Cliquez sur le nom d'une vidéo pour ouvrir la page Détails de la vidéo.
Créer un dossier
  1. Dans le menu de navigation de gauche, cliquez sur + Créer un dossier.
  2. Entrez un nom pour le dossier.
  3. Appuyez sur Entrée.
