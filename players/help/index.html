Joueurs

Le module Players vous permet de créer et de gérer des lecteurs vidéo.

Liste des lecteur s
  • Nom - Cliquez sur le lien du nom du lecteur pour modifier les détails du lecteur . L'ID du lecteur apparaît sous le nom du lecteur .
  • Mise à jour: date et heure de la dernière mise à jour
  • Version - Version du lecteur
  • Status - Statut du lecteur
    • - Indique que des modifications ont été apportées au lecteur mais qu'elles n'ont pas été publiées
    • - Indique que le lecteur est activé et que toutes les modifications apportées au lecteur ont été publiées
    • - Indique que le lecteur est inactif
Résumé du lecteur
  1. Sélectionnez le nom d'un lecteur pour voir le résumé à droite de la liste des lecteur s. Les informations incluent:
    • Nom du lecteur - Cliquez sur le lien du nom du lecteur pour modifier les détails du lecteur .
    • Created - Date de création du lecteur
    • Cliquez sur Aperçu du lecteur pour voir un aperçu de votre lecteur dans une fenêtre modale
    • Cliquez sur Code intégré et URL pour copier le code d'intégration du lecteur d'aperçu ou du lecteur publié afin de tester les modifications apportées à votre lecteur dans votre page HTML.
    • Player ID: identifiant unique du lecteur
    • Nom: nom du lecteur
    • Description - Brève description du lecteur
    • Status - Statut du lecteur
      • Active - Le lecteur est actif et prêt à être utilisé
      • Inactif - Le lecteur est désactivé
    • État publié: état publié du lecteur
      • Publié: le lecteur est publié et ne comporte aucune modification non publiée
      • Vous avez des modifications non publiées: vous avez apporté des modifications au lecteur ; les utilisateurs ne les verront pas tant que vous n'aurez pas publié le lecteur
Rechercher des lecteur s
  1. Dans Rechercher des lecteur s, commencez à taper pour rechercher des lecteur s dans votre compte. Les champs suivants sont recherchés:
    • Nom du lecteur
    • ID du lecteur
    • Brève description
Créer un lecteur
  1. Cliquez sur+Ajouter un lecteur .
  2. Saisissez le nom de votre lecteur .
  3. Saisissez une brève description pour votre lecteur .
  4. Cliquez sur Enregistrer.
Lorsque vous créez un nouveau lecteur, il est créé à l'aide de la dernière version du lecteur publiée par Brightcove.
Joueur en double
  1. Sélectionnez le lecteur que vous souhaitez dupliquer.
  2. Cliquez sur Dupliquer
  3. Saisissez le nom et la brève description du nouveau lecteur .
  4. Cliquez sur Enregistrer.
Supprimer un lecteur
  1. Sélectionnez le (s) lecteur (s) que vous souhaitez supprimer.
  2. Cliquez .
  3. Dans la boîte de dialogue Supprimer les lecteurs, confirmez en cliquant sur Supprimer.
