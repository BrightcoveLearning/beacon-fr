Télécharger des vidéos

Étapes

  1. Dans Beacon Studio, cliquez sur Toutes les vidéos dans le panneau de navigation.
  2. Cliquez sur Upload ou sur Upload Videos dans la navigation du panneau Le module Upload s'ouvre dans un nouvel onglet du navigateur.
  3. Sélectionnez un profil d'ingestion.
  4. Cliquez sur Rechercher des vidéos et parcourez et sélectionnez un ou plusieurs fichiers à télécharger ou glissez-déposez des fichiers du système de fichiers vers le module Upload.
  5. Le traitement des fichiers va commencer. Restez sur la page du module Upload (et n'actualisez pas la page) tant que toutes les vidéos n'ont pas le statut Processing Complete.
  6. Cliquez sur Afficher et modifier les détails de la vidéo pour afficher la vidéo et modifier les propriétés de la vidéo dans le module Médias.
