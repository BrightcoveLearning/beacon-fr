Prérequis

  • Vous avez téléchargé des vidéos sur votre compte Video Cloud
  • Vous avez créé une liste de lecteur manuelle ou intelligente

Étapes

  1. Dans la navigation principale, cliquez sur Media.
  2. Sélectionnez une liste de lecteur à publier:
    • Si la liste de lecteur apparaît dans la navigation latérale, cliquez dessus, puis cliquez sur Publier la liste de lecteur
    • Cliquez sur Toutes les listes de lecteur dans le menu de navigation de gauche, sélectionnez une liste de lecteur, puis cliquez sur Publier et intégrer > Publier sur le Web
  3. Dans la liste déroulante Sélectionner un lecteur , sélectionnez un lecteur à utiliser.
  4. Cliquez sur l'URL située sous le lecteur d'aperçu pour l'ouvrir dans un nouvel onglet du navigateur. Vérifiez que la liste de lecteur s'affiche et que les vidéos peuvent être lues.
  5. Copiez le code intégré et collez-le dans votre page Web.
