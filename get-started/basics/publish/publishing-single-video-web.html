Publier une seule vidéo sur le Web

Étapes

  1. Dans la navigation principale, cliquez sur Media.
  2. Cochez la case en regard de la vidéo à publier.
  3. Cliquez sur Publier sur le Web.
  4. Dans la liste déroulante Sélectionner un lecteur , sélectionnez un lecteur à utiliser.
  5. Cliquez sur l'URL située sous le lecteur d'aperçu pour l'ouvrir dans un nouvel onglet du navigateur. Vérifiez que la vidéo s'affiche et peut être lue.
  6. Copiez le code intégré et collez-le dans votre page Web.
