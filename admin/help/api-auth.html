Authentification API

Vous aurez besoin de jetons d'accès pour passer des appels vers les API Brightcove, sauf si vous avez activé l'authentification de base dans les paramètres du lecteur.

Ici, vous pouvez définir la portée de chaque jeton d'accès en enregistrant des applications.

Voir les applications
  • Nom - Cliquez sur ce lien pour afficher et modifier les détails de la demande.
  • Date de publication: date de création de l'application.
  • Supprimer: cliquez sur ce bouton pour supprimer l'application.
Enregistrer une nouvelle application
  1. Cliquez sur Enregistrer une nouvelle application.
  2. Ajouter/sélectionner des valeurs de champ:
    • Name: le nom de l'application
    • Description: description de l'application.
    • Sélectionner les comptes pour l'autorisation - Sélectionnez un ou plusieurs comptes pour lesquels vous souhaitez créer des jetons d'accès.
    • API Brightcove exposées: sélectionnez la ou les API et les autorisations à activer pour cette application. Pour des raisons de sécurité, Brightcove vous recommande de limiter la portée des autorisations.
  3. Cliquez sur Enregistrer.
Modifier la demande
  1. Cliquez sur le lien du nom de l'application.
  2. Modifier les valeurs des champs:
    • Name: le nom de l'application
    • Description: description de l'application.
    • Sélectionner les comptes pour l'autorisation - Sélectionnez un ou plusieurs comptes pour lesquels vous souhaitez créer des jetons d'accès.
    • API Brightcove exposées: sélectionnez la ou les API et les autorisations à activer pour cette application. Pour des raisons de sécurité, Brightcove vous recommande de limiter la portée des autorisations.
  3. Cliquez sur Enregistrer , Annuler ou Supprimer.
