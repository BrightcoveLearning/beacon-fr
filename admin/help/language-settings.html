Paramètres linguistiques

Définissez les langues associées aux métadonnées de votre vidéo.

Remarque: Si vous supprimez une langue utilisée, toutes les métadonnées associées à cette langue seront masquées. Pour restaurer les métadonnées associées, ajoutez à nouveau la langue.
Langue principale

La langue principale s'affiche initialement lorsque vous modifiez les propriétés de la vidéo dans le module Médias. Les métadonnées existantes seront associées à la langue principale.

Modifier la langue principale

  1. Cliquez sur la langue principale affichée pour la modifier.
  2. Commencez à taper votre langue pour voir une liste de valeurs.
  3. Sélectionnez une langue.
  4. Cliquez sur Enregistrer.
Langues supplémentaires

Après l'enregistrement d'une langue principale, des langues de métadonnées supplémentaires peuvent être ajoutées. Cela vous permet d'ajouter des métadonnées multilingues dans le module Médias.

Ajouter une langue

  1. Cliquez sur Ajouter une langue.
  2. Dans le champ de saisie Langue, commencez à saisir votre langue pour voir les valeurs.
  3. Sélectionnez une langue.
  4. Cliquez sur Enregistrer la langue.

Modifier la langue

  1. Cliquez sur une langue de la liste pour la modifier.
  2. Dans le champ de saisie Langue, commencez à saisir votre langue pour voir les valeurs.
  3. Sélectionnez une langue.
  4. Cliquez sur Enregistrer , Annuler ou .
