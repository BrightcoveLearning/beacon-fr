Paramètres de téléversement

Ces paramètres sont associés au téléchargement d'une vidéo dans le module Médias.

Réglages

  • Par défaut sur Inactif - Lorsque cette option est cochée, toutes les vidéos téléchargées sont désactivées
  • Champs obligatoires:
    • Requis pour l'activation - Incluez les valeurs de champ requises pour l'activation vidéo
    • Obligatoire pour le téléchargement - Incluez les valeurs de champ obligatoires lors du chargement d'une vidéo
  • Upload Additional Assets - Autoriser la génération d'images d'affiches et de vignettes lors du chargement d'une vidéo
  • Taille par défaut de l'image d'affiche (px) - La taille de l'image d'affiche par défaut utilisée lorsqu'elle n'est pas spécifiée dans le profil d'acquisition
  • Taille par défaut de l'image miniature (px): taille d'image miniature par défaut utilisée lorsqu'elle n'est pas spécifiée dans le profil d'acquisition
  • Mise en file d'attente des téléchargements
    • Autoriser les chargements simultanés: plusieurs fichiers sont téléchargés simultanément
    • Télécharger un fichier à la fois - Les fichiers sont téléchargés un par un (recommandé pour les fichiers volumineux)

Après avoir modifié les paramètres, cliquez sur Enregistrer.
