Pour les chaînes qui n'ont pas encore commencé à jouer

Modifier les détails des chaînes
  1. Cliquez .
  2. Modifiez les informations que vous souhaitez modifier, puis cliquez sur Mettre à jour Channel.
  3. Sélectionnez-en un, puis cliquez sur Supprimer pour les supprimer de votre compte
  4. Sélectionnez une configuration publicitaire, puis cliquez sur Dupliquer pour créer une copie de la configuration publicitaire
  5. Entrez un nom (en anglais uniquement, sans espaces ; ces caractères spéciaux sont autorisés: _.:/= + -).
  6. Si vous basculez Loop Cloud Playout Loop Cloud Playout on, vous pouvez également saisir une date/heure de fin facultative.
  7. Basculer Activez l'insertion publicitaire côté serveur (SSAI) ou vous pouvez exporter sur pour activer SSAI. L'option Live Input sera également supprimée.
  8. Cliquez Ajoutez des vidéos.
  9. Vous pouvez faire glisser des événements en direct pour réorganiser la liste. Vous pouvez également utiliser le Bouton Quick Edit pour modifier la durée d'un événement en direct.
  10. Développer Nouveau profil.
  11. Pour confirmer, cliquez sur Supprimer le profil.
  12. Pour supprimer, cliquez sur Supprimer.
  13. Cliquez Supprimez et oubliez.
  14. Dans Rechercher un utilisateur
