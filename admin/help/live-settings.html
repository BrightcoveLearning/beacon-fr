Réglages en direct

Gérez les listes de remplissage et les informations d'identification pour les flux Brightcove Live. Les configurations publicitaires SSAI sont gérées dans les paramètres des publicités côté serveur.

Ajouter un remplissage ardoise

Les listes de remplissage sont du contenu à diffuser lorsqu'il y a des lacunes dans une pause publicitaire en direct.

  1. Cliquez sur Ajouter un tableau de remplissage.
  2. Entrez le nom - un nom descriptif pour votre ardoise de remplissage.
  3. Entrez l'URL source - adresse complète de la liste de remplissage.
  4. Cliquez sur Enregistrer.
Supprimer la grille de remplissage
  1. Cliquez pour supprimer la palette de remplissage associée.
  2. Dans la boîte de dialogue Supprimer, cliquez sur Supprimer pour confirmer.
Ajouter des informations d'identification

Les informations d'identification en direct sont requises lors de l'utilisation du module Live ou de l'API Live.

  1. Pour les informations d'identification Live Video Cloud, commencez par les informations d'authentification API .
    Pour les informations d'identification S3, FTP ou RTMP, ignorez cette étape.
  2. Enter Label - un nom interne pour cette information d'identification
  3. Sélectionnez un type:
    • FTP - Utilisé pour enregistrer la ressource VOD sur un emplacement FTP (utilisation de l'API)
    • RTMP: utilisé pour créer des informations d'identification RTMP à utiliser par l'encodeur
    • S3 - Utilisé pour enregistrer l'actif VOD sur un compte Amazon S3 (utilisation de l'API)
    • Video Cloud - Utilisé pour enregistrer la ressource VOD sur un compte Video Cloud (utilisation de l'API)
  4. Entrez la clé publique/le nom d'utilisateur:
    • FTP - Saisissez le nom d'utilisateur du site FTP
    • RTMP - Entrez le nom d'utilisateur que l'encodeur utilisera pour s'authentifier
    • S3 - Entrez le nom d'utilisateur du compte S3
    • Video Cloud: saisissez l'ID client généré pour les informations d'identification d'authentification API
  5. Entrez la clé privée/le mot de passe:
    • FTP - Entrez le mot de passe du site FTP
    • RTMP - Entrez le mot de passe que l'encodeur utilisera pour s'authentifier
    • S3 - Entrez le mot de passe du compte S3
    • Video Cloud: saisissez le secret client généré pour les informations d'identification d'authentification API
  6. Cliquez sur Enregistrer.
Supprimer les informations d'identification
  1. Cliquez pour supprimer les informations d'identification associées.
  2. Dans la boîte de dialogue Supprimer, cliquez sur Supprimer pour confirmer.
