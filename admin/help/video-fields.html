Champs vidéo

Les champs vidéo sont des métadonnées associées à chaque vidéo de votre compte. Vous avez accès à un ensemble de champs standard et vous pouvez créer vos propres champs personnalisés.

Champs personnalisés

Champs créés par l'utilisateur.

  • Champ - Basculer entre afficher les noms internes et afficher les noms d'affichage pour les listes de champs personnalisées et standard.
  • Cliquez sur le nom d'un champ personnalisé pour le modifier.
  • Nom complet: nom du champ personnalisé. Caractères codés sur deux octets et espaces autorisés.
  • Nom interne: identifiant de l'API. Caractères alphanumériques en minuscules sans espaces.
  • Type: sélectionnez l'une des options suivantes:
    • Texte: champ de saisie de texte. Maximum de 1024 caractères codés sur un seul octet ou 512 caractères codés
    • Liste: liste déroulante de valeurs. Maximum de 150 valeurs, chaque valeur ne dépassant pas 100 caractères.
  • Description - Fournit des instructions utilisateur pour définir la valeur du champ. Il s'agit d'un indice pour chaque détail de vidéo sur la page Médias.
  • Make this Field Required - L'utilisateur doit saisir une valeur de champ personnalisée avant que la vidéo puisse être activée.
Ajouter un champ personnalisé

Vous pouvez créer vos propres champs personnalisés.

  1. Cliquez sur Ajouter un champ personnalisé
  2. Saisissez des valeurs de champ personnalisées.
  3. Cliquez sur Enregistrer le champ
Supprimer le champ personnalisé

Pour supprimer un champ personnalisé, contactez le support client .

Champs standard

Brightcove a fourni des champs.

  • Cliquez sur le nom d'un champ standard pour le modifier.
  • Description - Fournit des instructions utilisateur pour définir la valeur du champ. Il s'agit d'un indice pour chaque détail de vidéo sur la page Médias.
  • Make this Field Required - L'utilisateur doit saisir une valeur de champ personnalisée avant que la vidéo puisse être activée.
Besoin de plus de champs ?

Chaque compte reçoit un certain nombre de champs personnalisés. Si vous avez besoin de plus, contactez le support client .
