Éditeur avancé

Le profil d'acquisition définit le nombre et le type de rendus créés pour chaque téléchargement vidéo.

Actions
  • Utiliser par défaut pour le téléchargement - Définissez ce profil comme profil d'acquisition par défaut pour le téléchargement de vidéos
  • Utiliser par défaut pour le découpage social - Définissez ce profil comme profil d'ingestion par défaut pour le découpage social
  • Dupliquer - Pour créer un profil similaire, vous pouvez dupliquer ce profil et modifier les modifications nécessaires

données JSON

Les profils d'ingestion sont définis à l'aide de JSON, que vous pouvez modifier ici.

Si vous n'êtes pas à l'aise pour modifier JSON, Brightcove recommande de modifier les profils à l'aide de Simple Editor.
