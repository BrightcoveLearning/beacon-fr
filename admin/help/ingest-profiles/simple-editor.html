Éditeur simple

Utilisez l'éditeur simple pour créer et personnaliser vos profils d'acquisition.

Actions
  • Utiliser par défaut pour le téléchargement - Définissez ce profil comme profil d'acquisition par défaut pour le téléchargement de vidéos
  • Utiliser par défaut pour le découpage social - Définissez ce profil comme profil d'ingestion par défaut pour le découpage social
  • Dupliquer - Pour créer un profil similaire, vous pouvez dupliquer ce profil et modifier les modifications nécessaires
Données de profil
  • Nom: nom de votre profil d'acquisition
  • Description: texte descriptif de votre profil d'acquisition
  • Stocker les fichiers maîtres - Conserver une copie du fichier maître pour le retranscodage futur
  • Images - Spécifiez les dimensions des affiches et des vignettes
  • Capturez les images automatiquement: pendant le chargement, Video Cloud capture automatiquement les images de l'affiche et des vignettes à partir du point de 50 % de votre vidéo
  • Ne capturez pas d'images: Video Cloud ne capture pas d'images. Vous pouvez le faire ultérieurement dans le module Pages.
Rendus
  • Bouton Ajouter un rendu: ajoutez un rendu vidéo ou audio à votre profil d'acquisition ; affichez le lien Afficher la liste des formats associés pour voir les rendus disponibles
  • Lien Afficher la liste des formats associés: cliquez sur ce lien pour afficher les rendus vidéo et audio disponibles
  • Codec - Norme de compression vidéo
  • Nom: nom du format associé
  • Type: type de rendu ; valeurs: par défaut, personnalisé
  • Bitrate: quantité de bits de données vidéo transférés en une seconde, mesurée en kilobits par seconde (kbps)
  • Hauteur: hauteur de la vidéo, mesurée en pixels
