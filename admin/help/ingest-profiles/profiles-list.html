Profils d'ingestion

Les profils d'ingestion définissent le nombre et le type de rendus créés pour chaque téléchargement de vidéo.

Afficher le nom d'affichage/Afficher le nom de l'API
Cliquez pour basculer entre une liste de noms d'affichage de profil et des noms d'API de profil.
Afficher les champs
  • Nom et description du profil: cliquez sur le lien du nom pour modifier le profil
  • Version du profil (selon la configuration de votre compte):
    • Livraison dynamique - Stocke un petit ensemble de rendus et réalise l'emballage en temps réel
    • Context-Aware - Utilise l'apprentissage automatique pour optimiser le nombre et la qualité des rendus nécessaires
  • Type:
    • VOD standard: profil VOD par défaut fourni par Brightcove. Cela ne peut pas être modifié ou supprimé
    • Custom VOD: profil VOD personnalisé que vous avez créé
    • Standard Live: profil Live par défaut fourni par Brightcove. Cela ne peut pas être modifié ou supprimé
    • Custom Live: profil Live personnalisé que vous avez créé
  • Renditions: nombre de formats associés pouvant être créés
  • Afficher - Basculer pour masquer/afficher le profil sur la page de téléchargement
Ajouter un profil
  1. Développez Nouveau profil.
  2. Sélectionnez-en un:
    • Dynamic Delivery - Créer un nouveau profil de livraison dynamique
    • Live - Créer un nouveau profil Live
Supprimer le profil
  1. Cliquez pour supprimer le profil d'ingestion associé.
  2. Pour confirmer, cliquez sur Supprimer le profil.
