Rendus vidéo

Voici les rendus vidéo que vous pouvez inclure dans vos profils d'acquisition pour Dynamic Delivery.

  • Nom: nom du format associé
  • Type: type de rendu ; valeurs: par défaut, personnalisé
  • Codec - Norme de compression vidéo
  • Bitrate: quantité de bits de données vidéo transférés en une seconde, mesurée en kilobits par seconde (kbps)
  • Hauteur: hauteur de la vidéo, mesurée en pixels
  • Mode Aspect
    • Préserver - Préserver le rapport hauteur/largeur de la vidéo
    • Personnalisé - Sélectionnez le rapport hauteur/largeur de la vidéo
  • Profil: type de rendu
