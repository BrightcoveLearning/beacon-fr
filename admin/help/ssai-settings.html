Paramètres des publicités côté serveur

L'insertion publicitaire côté serveur (SSAI) de Brightcove intègre des publicités dans votre flux vidéo pour offrir une expérience de visionnage similaire à celle d'une télévision sans mise en mémoire tampon des publicités.

Actions
  • Supprimer: sélectionnez une ou plusieurs configurations d'annonces, puis cliquez sur Supprimer pour les supprimer de votre compte.
  • Dupliquer: sélectionnez une configuration d'annonce, puis cliquez sur Dupliquer pour créer une copie de la configuration publicitaire ; vous pouvez annuler ou modifier et enregistrer la nouvelle configuration
Afficher les configurations des annonces
  • Nom: nom de la configuration publicitaire ; cliquez sur le lien du nom pour afficher les détails de la configuration de l'annonce
  • Type - VOD ou Live
  • Réponse du serveur publicitaire: format de réponse de votre serveur publicitaire
  • Created: date de création de la configuration de l'annonce
Créer une configuration publicitaire
  1. Cliquez sur Create Ad Configuration pour développer le menu déroulant.
  2. Choisissez en un:
    • VOD - Vidéo à la demande
    • Live - Diffusion en direct
  3. Name: nom de la configuration de l'annonce
  4. Réponse du serveur publicitaire: format de réponse de votre serveur publicitaire
    • Règles publicitaires DFP - Format propriétaire pour DFP/Google for Publishers
    • VMAP - Playlist de plusieurs annonces vidéo
    • XML intelligent - Format propriétaire pour FreeWheel
    • VASTE 3.0 - Modèle de diffusion d'annonces vidéo numériques
  5. Ad Tag: l'URL du serveur de votre balise publicitaire
  6. Désactiver les balises côté serveur: ces balises émettent des événements à partir du serveur pour mesurer les impressions publicitaires et signaler les erreurs publicitaires ; si elles sont désactivées, les événements ne seront pas déclenchés par le serveur, mais seront inclus dans la réponse publicitaire
  7. Cliquez sur Enregistrer.
Modifier la configuration des annonces
  1. Cliquez sur le lien du nom de la configuration publicitaire.
  2. Modifiez les valeurs de configuration des annonces.
  3. Cliquez sur Enregistrer.
