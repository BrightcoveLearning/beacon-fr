Paramètres du lecteur

Les paramètres du lecteur définissent le type d'authentification utilisé avec les API Brightcove.

Authentification basique

    • Activer - Cochez la case pour permettre à vos lecteur s d'utiliser l'authentification de base par nom d'utilisateur et mot de passe pour accéder à l'API Player Management et Delivery System API
    • Désactiver - Décochez la case pour désactiver l'authentification de base ; dans ce cas, l'authentification API doit être utilisé
  2. Cliquez sur Enregistrer.
