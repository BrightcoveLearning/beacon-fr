Paramètres du compte

Définissez le fuseau horaire de reporting par défaut pour votre compte.

  1. Sélectionnez votre fuseau horaire actuel.
    • Il est utilisé pour calculer les limites journalières pour les analyses
    • Les modifications ne concernent que les données à l'avenir
  2. Cliquez sur Enregistrer.
