Informations sur le compte

Affichez et modifiez les informations du compte Brightcove Beacon.

Afficher les informations du compte

  • Account Name: nom descriptif du compte
  • Account ID - ID de compte attribué
  • Adresse - Adresse postale du compte
  • Téléphone - Numéro de téléphone du contact
  • Max Users: nombre maximal d'utilisateurs autorisés pour le compte. Gérer les utilisateurs dans la section Gestion des utilisateurs
Modifier les informations du compte
  1. Cliquez sur Modifier les informations du compte.
  2. Mettre à jour les valeurs des champs
    Remarques:
    • L'ID de compte ne peut pas être modifié
    • Le nombre maximum d'utilisateurs est basé sur votre type de compte
  3. Cliquez sur Enregistrer.
