Présentation

Lorsque vous créez des événements en direct à l'aide du module Live, vous devez sélectionner un profil d'acquisition Live qui détermine le nombre et la qualité des rendus créés par Brightcove Live. Cette rubrique fournit des informations détaillées sur chacun des profils d'acquisition Standard Live.

Remarques:
  • Pour les tâches créées via l'API Live, vous ne pouvez actuellement pas spécifier de profil d'acquisition, mais vous devez définir les caractéristiques de sortie.
  • Ces profils sont utilisés spécifiquement lors de la création d'événements en direct. Ils ne sont pas utilisés lors du chargement de vidéos sur Video Cloud.

Il est également possible de créer vos propres profils d'acquisition Live personnalisés. Pour plus d'informations, consultez la section Création de profils Live Ingest personnalisés.

Données du document

Remarques:
  • Cette page est créée à partir des données en direct de l'API ; le chargement complet peut prendre une minute.
  • Les profils standard ne peuvent pas être modifiés. Si vous souhaitez une version modifiée, vous devez créer un profil personnalisé.
