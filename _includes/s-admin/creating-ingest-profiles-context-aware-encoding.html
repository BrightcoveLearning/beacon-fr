Les profils d'ingestion contiennent les paramètres de transcodage qui seront utilisés lorsque les vidéos seront téléchargées à l'aide du module Upload ou de l'API Dynamic Ingest. L'éditeur de profil d'acquisition qui s'affiche change en fonction des options de transcodage configurées pour le compte.

Remarques:
  • La possibilité de créer et de modifier des profils d'acquisition n'est disponible que pour les éditeurs disposant de comptes Enterprise et Professional.
  • Si vous souhaitez pouvoir retranscoder les vidéos ingérées avec un profil personnalisé, vous devez archiver le master numérique (comme le font tous les profils standard). Consultez Archiving Masters pour plus de détails sur la procédure à suivre.

Pour accéder à la page d'administration des profils d'ingestion, cliquez sur le lien Admin, puis sur Ingest Profiles. Une liste des profils d'acquisition s'affiche.

Remarque: Pour accéder à la page Profils d'acquisition, vous devez être un utilisateur {{site.product}} avec le rôle d'administrateur.

Votre compte Brightcove sera configuré avec une ou plusieurs options de transcodage. Tous les comptes Brightcove sont configurés avec un ensemble par défaut de profils d'acquisition qui ne peuvent être ni modifiés ni supprimés. Ceux-ci sont marqués d'un type de VOD standard dans le studio. La version du profil affiche le type de transcodage utilisé avec le profil. Pour plus de détails sur les rendus fournis avec les profils d'acquisition fournis par Brightcove, consultez le document approprié:

  • Livraison dynamique
  • Context Aware

Création d'un nouveau profil adapté au contexte

La page d'administration des profils d'ingestion peut être utilisée pour créer de nouveaux profils d'ingestion. Les profils d'ingestion sont définis à l'aide de JSON.

Pour créer un nouveau profil d'acquisition tenant compte du contexte:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Cliquez sur Nouveau profil > Context Aware. L'éditeur de profil s'ouvre.
  4. Entrez un nom, un nom complet et une description pour le profil.
  5. Décidez si vous souhaitez stocker les fichiers maîtres. Le stockage du fichier maître conserve une copie du fichier principal et permet de retranscoder la vidéo à l'avenir sans qu'il soit nécessaire de la télécharger à nouveau. Pour plus d'informations sur les fichiers maîtres d'archivage, consultez le document Masters d'archivage.
  6. Sélectionnez un paramètre pour les images:
    • Capturez les images automatiquement au moment du chargement: Brightcove capture les affiches et les vignettes lorsque la vidéo est téléchargée. Les images sont capturées à 50 % dans la vidéo.
    • Ne capturez pas d'images ; je téléchargerai ou capturerai mes propres images - Les affiches et les vignettes seront capturées ou téléchargées à l'aide du module Médias.
  7. Saisissez la largeur et la hauteur des images de l'affiche et des vignettes.
  8. Utilisez la liste déroulante Codec pour sélectionner le codec vidéo à utiliser. Vous pouvez choisir entre H264 et HEVC, ou utiliser un profil de codec mixte qui créera des rendus utilisant les deux codecs (H264, HEVC) . :
  9. Sélectionnez une résolution minimale et une résolution maximale en faisant glisser le curseur. Cette plage sera utilisée lors de la création de rendus.
  10. Sélectionnez un débit binaire maximal. Les rendus supérieurs à ce débit binaire ne seront pas créés quelle que soit la complexité du contenu. Un débit binaire inférieur peut aider à contrôler les coûts liés au stockage et à la bande passante.
  11. Sélectionnez les résolutions de sortie requises. Les résolutions requises ne seront pas créées si elles sont supérieures à la résolution du fichier source.
  12. Réglez le contrôle qualité de l'encodage pour régler la qualité vidéo et l'utilisation de la bande passante. Si vous êtes plus préoccupé par l'utilisation de la bande passante et que vous souhaitez optimiser la consommation de bande passante la plus faible, déplacez le curseur vers la gauche. Si vous êtes plus préoccupé par la qualité et que vous êtes d'accord avec l'idée d'utiliser davantage de bande passante pour offrir une meilleure qualité d'image, déplacez le curseur vers la droite.
  13. Sélectionnez un rendu MP4 progressif à créer. Au moins un rendu MP4 est requis pour publier des vidéos sur des comptes de réseaux sociaux à l'aide du module Social ou pour autoriser le téléchargement de vidéos. Sélectionnez Aucun si aucun format associé MP4 ne doit être créé.
  14. Sélectionnez les rendus audio à créer.
  15. Cliquez sur Enregistrer pour enregistrer le nouveau profil.
Notez que si vous avez besoin de rendus avec des résolutions spécifiques, il est possible de l'exiger dans le profil, mais pas dans Studio. Au lieu de cela, vous devrez utiliser l'API Ingest Profiles.
Notez qu'il n'est pas recommandé d'exiger des rendus, car cela réduit l'efficacité du codage sensible au contexte. Si vous avez plusieurs résolutions requises, le CAE n'est peut-être pas la bonne option pour vous.

Duplication des profils d'ingestion

Si vous devez créer un profil similaire à un profil existant, vous pouvez dupliquer un profil existant, puis le modifier pour apporter les modifications nécessaires.

Pour dupliquer un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Cliquez sur le profil d'acquisition pour le dupliquer.
  4. Cliquez sur Dupliquer
  5. Utilisez l'éditeur de profil pour modifier le nom et la définition du profil.
  6. Cliquez sur Enregistrer.

Modification des profils d'ingestion

La page d'administration des profils d'ingestion peut être utilisée pour modifier les profils d'ingestion. Notez que l'ensemble de profils d'acquisition par défaut (marqué Standard VOD) ne peut pas être modifié.

Pour modifier un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Cliquez sur le profil d'acquisition pour le modifier.
  4. Utilisez l'éditeur de profil pour modifier la définition du profil.
  5. Cliquez sur Enregistrer.

Suppression des profils d'acquisition

La page d'administration des profils d'ingestion peut être utilisée pour supprimer les profils d'acquisition qui ne sont plus nécessaires. Notez que l'ensemble de profils d'acquisition par défaut (marqué Standard VOD) ne peut pas être supprimé, mais ils peuvent être masqués s'ils ne sont pas nécessaires.

Pour supprimer un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles. La page d'administration des profils d'ingestion s'ouvre et affiche les profils d'acquisition qui ont été créés pour votre compte.
  3. Cliquez sur l'icône de suppression () en regard du profil d'acquisition ou cliquez sur le nom du profil, puis sur Supprimer.
  4. Confirmez la suppression en cliquant sur Supprimer le profil.

Modification du profil d'acquisition par défaut pour le chargement

Tous les comptes Brightcove disposent d'un profil d'acquisition par défaut pour le chargement. Le profil d'acquisition par défaut de chargement est indiqué en regard du nom du profil.

Pour modifier le profil par défaut du compte:

  1. Ouvrez le module Admin.
  2. Cliquez sur le lien Ingest Profiles. La page d'administration des profils d'ingestion s'ouvre et affiche les profils d'acquisition qui ont été créés pour le compte.
  3. Cliquez sur le nom du profil d'acquisition du profil que vous souhaitez définir comme profil par défaut.
  4. Cliquez sur Chargement en regard du lien Utiliser par défaut en haut de la page.

Masquer et afficher les profils d'ingestion

Les profils d'acquisition peuvent être masqués en cliquant sur le curseur associé au profil sur la position Masquer. Cela est utile pour masquer les profils par défaut qui ne sont peut-être pas nécessaires.

Les profils d'ingestion masqués n'apparaîtront pas dans la liste Profil d'acquisition du module Upload.

Prise en charge DRM dans les profils

Contactez le support Brightcove pour définir les paramètres par défaut pour l'empaquetage DRM de vos vidéos. Il n'est pas nécessaire de spécifier un package DRM dans vos profils d'acquisition. Si votre compte est activé pour les DRM et que vous souhaitez diffuser certains titres sans DRM, modifiez les propriétés de la vidéo dans le module Médias et définissez l'indicateur DRM pour la vidéo sur Pas de DRM.

Remarque: Pour plus d'informations sur la création d'un lecteur Brightcove compatible DRM, consultez la rubrique Plug-in DRM.

Utilisation de l'éditeur avancé

Les profils d'ingestion sont définis à l'aide de JSON. Pour afficher la syntaxe JSON d'un profil, modifiez un profil, puis cliquez sur l'onglet Éditeur avancé. La représentation JSON du profil s'affiche.

Bien que le JSON puisse être modifié ici, Brightcove recommande de modifier les profils à l'aide de Simple Editor afin de réduire le risque d'erreurs. Pour plus d'informations sur les attributs disponibles, consultez la rubrique Ingest Profiles API Fields Reference for Dynamic Delivery and Context Aware Encoding.
