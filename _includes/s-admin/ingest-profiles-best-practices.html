Combien de profils dois-je avoir ?

Vous pouvez définir jusqu'à 200 profils personnalisés. Toutefois, vous devez généralement n'en créer que le nombre dont vous avez besoin, afin d'éviter toute confusion lors de la sélection de profils pour les tâches d'ingestion.

Comment gérer mes profils ?

Vous pouvez utiliser la page Gestion des profils d'ingestion de Studio pour gérer vos profils d'ingestion. Diverses options sont disponibles dans cette interface, notamment:

  • Définition de votre profil par défaut
  • Création de nouveaux profils
  • Clonage à partir de profils existants
  • Modification des profils personnalisés
  • Suppression des profils inutiles

Si vous avez besoin d'un accès par programmation à vos profils d'acquisition, vous pouvez créer, lire, mettre à jour et supprimer des profils personnalisés à l'aide de l'API REST Profiles Ingest Profiles. Notez que cela nécessite que vous définissiez des informations d'identification OAuth.

Comment puis-je mettre à jour mes profils ?

Reportez-vous à la section ci-dessus.

Combien de rendus un profil peut-il comporter ?

Un profil de diffusion dynamique peut comporter jusqu'à 37 rendus. Notez que pour les profils de livraison dynamique personnalisés, vous ne pouvez sélectionner que les formats associés prédéfinis actuellement disponibles.

Si vous créez un profil Context Aware Encoding personnalisé, min_renditions a une valeur minimale de 1 et max_renditions une valeur maximale de 15.

Notez que les rendus inutiles augmenteront vos coûts de stockage géré. Par conséquent, vous ne devez pas créer de rendus peu susceptibles d'être lus compte tenu du type de vidéos que vous publiez et des publics et appareils que vous ciblez.

Les profils standard de votre compte sont conçus pour offrir de bonnes expériences de visionnage sur les bandes passantes classiques des spectateurs et sur les appareils les plus couramment utilisés.

Nommer les profils personnalisés

Les noms de profil personnalisés ne doivent pas inclure d'espaces ni de caractères spéciaux. Utilisez des lettres et des chiffres, et remplacez les espaces par des tirets (-) ou des traits de soulignement (_).

Pour éviter les erreurs des utilisateurs, essayez de nommer les profils de manière à indiquer clairement aux personnes qui téléchargent des vidéos à quoi le profil doit être utilisé.

Quel profil dois-je définir par défaut ?

Tout profil standard ou personnalisé peut être défini par défaut dans Studio ou via l'API Ingest Profile. La valeur par défaut ne sera utilisée que si l'utilisateur ne spécifie pas de profil particulier (soit dans le module Upload, soit à l'aide de l'API Dynamic Ingest).

Nous vous recommandons de définir le profil par défaut sur un profil qui ne serait pas un très mauvais choix si l'utilisateur oublie de spécifier un profil particulier.
