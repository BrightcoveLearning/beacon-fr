Le partage de médias permet aux éditeurs de télécharger des vidéos une seule fois, puis de les partager sur d'autres comptes. Il peut s'agir de vos propres comptes, de comptes d'entreprise ou de comptes d'autres éditeurs. Les vidéos partagées peuvent ensuite être utilisées indépendamment dans les autres comptes {{site.product_short}}. Les paramètres de partage multimédia sont utilisés pour gérer les relations de partage entre les comptes.

Remarque: Le partage de médias n'est disponible que pour les clients {{site.product_short}} Enterprise. Si vous souhaitez mettre à niveau votre compte {{site.product_short}}, veuillez contacter Brightcove pour plus d'informations.

Le partage de médias peut également être implémenté par programmation à l'aide de l'API CMS. Pour plus d'informations, consultez la section Partage de médias avec l'API CMS.

Cette rubrique utilisera la terminologie suivante:

  • Master - Le compte qui a créé la vidéo originale. Le Master est propriétaire du contenu et est responsable de la configuration, de la gestion et de la fourniture du contenu aux affiliés.
  • Affilié - Le compte qui reçoit la vidéo. L'Affilié peut accepter le contenu qui lui est partagé par un Master.

Activation du partage multimédia

Pour activer le partage multimédia, contactez le support client de Brightcove et demandez-lui d'activer le partage multimédia pour votre compte.

Ajout d'affiliés

Pour partager des vidéos sur un autre compte {{site.product_short}}, un compte d'affilié doit être ajouté. Un affilié est un compte sur lequel vous souhaitez partager des vidéos. Pour ajouter un compte d'affilié, procédez comme suit:

  1. Connectez-vous à {{site.product}} en tant qu'administrateur.
  2. Ouvrez le module Admin, puis Media Sharing. Notez que vous devez être administrateur de compte pour accéder au menu ADMIN.
  3. La section Gérer les affiliés - Comptes avec lesquels je partage des vidéos affiche la liste de tous les éditeurs affiliés avec lesquels vous avez établi une relation de partage de médias.
  4. Pour ajouter un nouvel affilié, entrez son {{site.product_short}} ID de compte dans la zone de texte ID de compte, puis cliquez sur Ajouter un affilié.

Le compte sera ajouté au tableau avec un statut de relation en attente. Le compte d'affilié recevra une notification de partage de médias et pourra accepter ou rejeter la relation. Si l'affilié accepte la relation de partage de médias, le statut de la relation sera Accepté.

Mise en correspondance des champs

Pour exiger que les comptes affiliés aient les mêmes champs personnalisés que le compte principal, cochez la case Appliquer la correspondance des champs personnalisés, puis cliquez sur Mettre à jour les règles. Lorsque l'option Appliquer la correspondance des champs personnalisés est cochée, les comptes d'affiliation doivent avoir les mêmes champs personnalisés que le compte principal, faute de quoi les actions échoueront.

Remarques

  • La mise en correspondance des champs personnalisés ne fera que vérifier que les noms des champs personnalisés internes des deux comptes correspondent. Le type de champ (liste ou texte) n'a pas d'importance.

  • Si la chaîne définie par le compte principal applique la correspondance des champs personnalisés, puis partage une vidéo dont la valeur n'est pas autorisée par le compte d'affilié, cette tentative de partage échouera.

    Si la chaîne définie par le compte principal n'applique pas la correspondance des champs personnalisés, puis partage une vidéo dont le champ personnalisé a une valeur non autorisée par le compte affilié, la tentative de partage fonctionnera, mais le champ avec la mauvaise valeur ne sera pas inclus dans la copie affiliée de la vidéo .

  • Remarque: Si une vidéo partagée dans le compte principal est configurée avec le filtrage géographique, les comptes d'affiliation doivent également être activés pour le filtrage géographique, faute de quoi le partage échouera. Tout filtrage géographique configuré sur la vidéo partagée sera également appliqué au (x) compte (s) d'affilié (s). Pour activer un compte pour le filtrage géographique, veuillez contacter le support Brightcove.

Supprimer des affiliés

Pour supprimer un compte d'affilié, localisez le compte à supprimer et cliquez sur l'icône Supprimer (). Une boîte de dialogue apparaît pour confirmer que vous souhaitez mettre fin à la relation de partage.

Cliquez sur Fin du partage pour mettre fin à la relation de partage.

Approuver/rejeter les relations de partage des médias

Si un autre compte {{site.product_short}} a ajouté votre compte en tant qu'affilié (souhaite partager des vidéos sur votre compte {{site.product_short}}), un message s'affichera lorsque vous accéderez à la page Admin > Partage multimédia.

Si l'option Appliquer la correspondance des champs personnalisés est cochée, le compte d'affilié doit avoir les mêmes champs personnalisés que le compte principal, faute de quoi les actions échoueront.

Cliquez sur Rejeter pour rejeter la relation de partage de médias. Sinon, sélectionnez un mode d'acceptation:

  • Automatique - Acceptez toujours les vidéos partagées de ce compte principal. Les vidéos partagées apparaîtront dans la liste des vidéos du module Médias.
  • Manuel - Passez en revue les vidéos partagées de ce compte principal avant de les ajouter à la médiathèque. Le module Médias affichera un élément Partages en attente dans le menu de navigation de gauche.

Cliquez sur Approuver pour approuver la relation de partage de médias. Une fois la relation approuvée, elle apparaît dans la section Gérer les maîtres de la page.

Cliquez sur Afficher le contrat pour afficher les paramètres de partage multimédia. Le curseur Mode d'acceptation peut être utilisé pour modifier le mode d'acceptation. Pour supprimer la relation de partage, cliquez sur l'icône Supprimer ().
