Informations sur le compte

L'option Informations sur le compte vous permet de modifier le nom du compte et l'adresse postale et peut être utilisée pour cloner des comptes. Pour plus d'informations, consultez la section Gestion des informations de compte.

Gestion des utilisateurs

L'option Gestion des utilisateurs vous permet de gérer les utilisateurs autorisés à utiliser Studio. Pour plus d'informations sur la gestion des utilisateurs, consultez la section Gestion des utilisateurs.

Paramètres du compte

Les paramètres d'analyse permettent d'accéder aux paramètres liés au compte. Le fuseau horaire de reporting par défaut est utilisé pour calculer les limites journalières pour les données de reporting. La modification du paramètre de fuseau horaire des rapports n'affecte que les données à venir, et les modifications ne sont pas appliquées rétroactivement. Si vous modifiez ce paramètre, vous pouvez voir un point plat ou un pic dans vos données d'analyse lorsque vous examinez les limites des jours où la modification a été appliquée. Les mises à jour de ce paramètre peuvent ne pas prendre effet immédiatement et les données d'analyse peuvent continuer à faire référence à l'ancien paramètre jusqu'à ce que le système traite les modifications.

Si le champ Activer le type de contenu vidéo est activé, le champ Type de contenu vidéo s'affiche sur la page d'accueil du module Médias. Il permet également d'attribuer un type de contenu vidéo aux vidéos sur la page des propriétés de la vidéo. Pour plus d'informations sur la gestion des utilisateurs, consultez la section Présentation des types de contenu vidéo.

Paramètres de téléversement

Par défaut, Inactive - Lorsque cette option est cochée, toutes les vidéos téléchargées sont désactivées. Cette option est ignorée si l'utilisateur a accès au module Media. Cela peut être utilisé pour permettre à un ensemble d'utilisateurs de télécharger des vidéos, mais ils resteront inactifs jusqu'à ce que quelqu'un d'autre les approuve et les active.

Champs obligatoires - Sélectionnez une option pour la manière dont les champs obligatoires sont traités:

  • Obligatoire pour l'activation: les champs obligatoires doivent contenir des valeurs pour que la vidéo puisse être activée.
  • Obligatoire pour le téléchargement - Les champs obligatoires doivent contenir des valeurs pour que la vidéo puisse être traitée et téléchargée à l'aide du module Upload.

Upload Additional Assets (Charger des ressources supplémentaires): lorsque cette option est cochée, les images des affiches et des vignettes peuvent être téléchargées lorsque des vidéos La boîte de dialogue d'état du chargement s'affiche et permet de sélectionner des affiches et des vignettes à télécharger. Pour plus d'informations, consultez la section Chargement de vidéos à l'aide du module de téléchargement.

Default Image Dimensions (Dimensions d'image par défaut): ces valeurs peuvent être utilisées pour définir les tailles d'image par défaut des affiches et des vignettes. En configurant les tailles dans le cadre des paramètres de téléchargement, vous pouvez éviter de devoir définir les dimensions des images dans vos profils d'acquisition. Lorsque ces valeurs sont configurées et que vos profils d'acquisition ont également des tailles d'image, {{site.product_short}} utilisera le processus suivant pour déterminer les dimensions à utiliser pour différents types d'acquisition d'images:

  1. Lorsque vous chargez une vidéo à l'aide du module Upload, {{site.product_short}} vérifie d'abord si des dimensions d'image sont définies dans le profil d'acquisition sélectionné. Si le profil d'acquisition comporte des dimensions d'image, celles-ci seront utilisées. Sinon, {{site.product_short}} vérifiera les dimensions dans l'ordre suivant:
    • Utiliser les dimensions d'image spécifiées dans le profil d'acquisition par défaut du compte
    • Utiliser les dimensions de l'image spécifiées dans les paramètres de téléchargement
    • Utiliser les dimensions d'image par défaut définies par {{site.product_short}}
  2. Si vous téléchargez une affiche ou une image miniature à partir de la page de détails de la vidéo, ou si vous capturez une image à partir de la page de détails de la vidéo, les dimensions d'image spécifiées dans le profil d'acquisition par défaut du compte seront utilisées. Si le profil d'acquisition par défaut ne comporte pas de dimensions d'image, {{site.product_short}} vérifiera les dimensions dans l'ordre suivant:
    • Utiliser les dimensions de l'image spécifiées dans les paramètres de téléchargement
    • Utiliser les dimensions d'image par défaut définies par {{site.product_short}}

Si vous souhaitez toujours utiliser les dimensions d'image spécifiées dans les paramètres de chargement, Brightcove vous recommande de définir ces valeurs de manière appropriée, puis de supprimer toutes les tailles d'image de vos profils d'acquisition.

Upload Queueing - Sélectionnez une option pour la manière dont les téléchargements de fichiers sont gérés:

  • Autoriser les chargements simultanés: plusieurs fichiers sont téléchargés simultanément
  • Upload one file at a time - Les fichiers sont téléchargés un par un ; ceci est recommandé lorsque des fichiers volumineux sont téléchargés

Profils d'ingestion

L'option Ingest Profiles vous permet de gérer les profils d'acquisition de votre compte {{site.product_short}}. Pour plus d'informations sur la gestion des profils d'ingestion, consultez la section Gestion des profils d'ingestion.

Champs vidéo

L'option Champs vidéo vous permet de marquer les champs vidéo standard selon vos besoins et vous permet également d'afficher, de créer et de modifier des champs de métadonnées personnalisés pour votre compte. Pour plus d'informations sur la configuration des champs vidéo, consultez la section Création de champs de métadonnées personnalisés.

Paramètres linguistiques

Les paramètres de langue permettent de définir une langue principale, puis d'ajouter et de gérer des langues de métadonnées supplémentaires. Pour plus d'informations sur la configuration des paramètres linguistiques, consultez Configuration des paramètres linguistiques.

Partage de médias

L'option Partage multimédia vous permet de télécharger des vidéos une seule fois, puis de les partager sur d'autres comptes {{site.product_short}}. Pour plus d'informations sur le partage de médias, consultez la section Gestion des paramètres de partage multimédia.

Paramètres des publicités côté serveur

Les paramètres des publicités côté serveur vous permettent de gérer les configurations des publicités en direct et VOD côté serveur. Pour plus d'informations, consultez la section Configuration des paramètres des publicités côté serveur.

Paramètres du lecteur

Authentification de base: permet d'utiliser l'authentification de base par nom d'utilisateur et mot de passe pour accéder aux API Player Management and Delivery System. Si cette option n'est pas activée, l'authentification par API doit être utilisée.

Mappage des lecteur s: permet de créer des mappages de lecteur s à utiliser dans l'outil de migration dirigée. L'outil de migration dirigée permet de spécifier qu'un lecteur Brightcove doit être chargé au lieu d'un lecteur intelligent.

Réglages en direct

Les paramètres Live vous permettent de gérer les paramètres de Brightcove Live. Pour plus d'informations, consultez la section Gestion des paramètres Live.

Intégrations

Les intégrations vous permettent d'intégrer Zoom et de diffuser une réunion Zoom à l'aide de Brightcove Live. Pour plus d'informations, consultez la section Intégration d'une réunion Zoom dans Brightcove Live.

Authentification API

L'option Authentification API vous permet de créer et de gérer les informations d'identification d'API. Pour plus d'informations sur la gestion des informations d'identification d'API, consultez la section Gestion des informations d'authentification

facturation

L'option Facturation vous permet de consulter les informations de facturation du compte. Pour plus d'informations sur la facturation, consultez la section Affichage des informations de facturation.
