Les droits de lecteur sont utilisés pour restreindre la lecteur des vidéos. Une fois les droits de lecteur créés, ils peuvent être attribués aux vidéos à l'aide du module Médias.

Pour accéder à la page d'administration des droits de lecteur, ouvrez le module Admin, puis Droits de lecteur. Une liste des droits de lecteur s'affiche.

Remarque: Pour accéder à la page Droits de lecteur, vous devez être un utilisateur {{site.product}} avec le rôle d'administrateur.
Les droits de lecteur sont attribués aux vidéos dans la section Restrictions de lecteur de la page des propriétés de la vidéo du module Médias.

Les droits de lecteur dont l'état est Inactif ne peuvent pas être attribués aux vidéos dans le module Médias.

Création de droits de lecteur

Pour créer des droits de lecteur, procédez comme suit.

  1. Connectez-vous à {{site.product}} en tant qu'administrateur.
  2. Ouvrez le module Admin, puis les droits de lecteur. Notez que vous devez être administrateur de compte pour accéder au menu ADMIN.
  3. Cliquez sur Nouveaux droits de lecteur.
  4. Saisissez un nom de droits de lecteur.
  5. Si vous cochez Autoriser la lecteur droite sans jeton, les demandes de lecteur peuvent être effectuées sans jeton Web JSON (JWT).
    Remarque: Désélectionner cette option n'affectera pas les restrictions définies pour ce droit de lecteur, cependant, la demande de lecteur nécessitera un JWT valide qui sera défini par programme au niveau du lecteur. Pour plus d'informations, consultez le document du développeur Using Playback Authorization DRM with the Brightcove Player.
  6. Cochez Utiliser par défaut pour que ce soit le droit de lecteur par défaut. Cela appliquera les restrictions de droits de lecteur configurées aux vidéos mises en ligne.
    Remarque: Le droit de lecteur par défaut n'est pas attribué si une vidéo est retranscodée.
  7. Cliquez sur Enregistrer.
Remarque: Le nombre maximum de droits de lecteur que vous pouvez avoir sur un compte est de 100. Si vous devez augmenter la limite, contactez le support Brightcove.

Cliquez sur un lien de restriction dans le menu de navigation de gauche pour ajouter des restrictions à une lecteur à droite:

  • Geographic
  • Domaine
  • Proxy
  • Heure

Modification des droits de lecteur

Pour modifier les droits de lecteur, procédez comme suit:

  1. Cliquez sur une lecteur à droite.
  2. Apportez les modifications requises et cliquez sur Enregistrer.

Duplication des droits de lecteur

Pour dupliquer un droit de lecteur, procédez comme suit:

  1. Cliquez sur une lecteur à droite.
  2. Cliquez sur Dupliquer
  3. Apportez les modifications requises et cliquez sur Enregistrer.

Activation/désactivation des droits de lecteur

Les droits de lecteur ne peuvent pas être supprimés, mais ils peuvent être désactivés afin qu'ils ne puissent pas être attribués aux vidéos dans le module Médias. Pour activer/désactiver les droits de lecteur, procédez comme suit:

  1. Cliquez sur une lecteur à droite.
  2. Cliquez sur Activer pour activer un droit de lecteur ou sur Désactiver pour désactiver un droit de lecteur. Les droits de lecteur désactivés ne peuvent pas être attribués aux vidéos du module Médias.

Attribution de restrictions géographiques

Les restrictions géographiques vous permettent de limiter la lecteur des vidéos en fonction du pays, du code postal, des adresses IP et de la zone de marché désignée. Pour attribuer des restrictions géographiques à un droit de lecteur, procédez comme suit:

  1. Cliquez sur une lecteur à droite.
  2. Dans la navigation de gauche, cliquez sur Restrictions géographiques.

La règle globale peut être définie sur Autoriser tout ou Tout bloquer.

  • Autoriser tout
    • Utilisé lorsque vous souhaitez mettre sur liste noire un petit nombre de groupes qui n'auront pas accès
    • Sélectionnez cette option pour autoriser l'affichage pour tous, puis ajoutez une liste de pays sur liste noire, de codes postaux, d'adresses IP et de DMA qui ne peuvent pas être consultés
  • Bloquer tout
    • Utilisé lorsque vous souhaitez mettre sur liste blanche un petit nombre de groupes qui auront accès
    • Sélectionnez cette option pour bloquer l'affichage pour tous, puis ajoutez une liste de pays, de codes postaux, d'adresses IP ou de DMA sur liste blanche qui sont autorisés à être affichés

Des restrictions géographiques peuvent être ajoutées pour autoriser/empêcher le visionnage en fonction des éléments suivants:

  • Pays
  • Codes postaux
  • Restrictions IP
  • Zone de marché désignée (DMA)

Restreindre l'accès aux pays

Les droits de lecteur par pays peuvent être utilisés pour limiter le visionnage à des pays spécifiques.

  1. Sélectionnez les pays dont l'affichage sera interdit (liste noire) ou autorisé (liste blanche). Au fur et à mesure que vous tapez, la liste de sélection sera filtrée.
  2. Cliquez sur Enregistrer.

Restreindre l'accès aux codes postaux

Les droits de lecteur du code postal peuvent être utilisés pour limiter l'affichage à des codes postaux spécifiques.

  1. Entrez une liste de codes postaux dont l'affichage ne sera pas autorisé (sur liste noire) ou autorisé (sur liste blanche). Les codes postaux doivent être précédés d'un code pays à deux lettres majuscules et d'un tiret, par exemple US-90045.

Il est également possible de télécharger un fichier .csv contenant des codes postaux. Cliquez sur Upload CSV pour parcourir et sélectionner un fichier à télécharger.

Remarques:
  • Jusqu'à 5 000 codes postaux sont pris en charge.
  • Le chargement d'un fichier .csv remplacera le contenu du champ des codes postaux.

Restreindre l'accès aux adresses IP

Les droits de lecteur IP peuvent être utilisés pour limiter l'affichage à une plage d'adresses IP.

  1. Entrez une plage d'adresses IP dont l'affichage sera interdit (sur liste noire) ou autorisé (sur liste blanche). La notation par bloc CIDR est également prise en charge.
  2. Cliquez sur Enregistrer.

Restreindre l'accès aux DMA

Une zone de marché désignée (DMA), également appelée marché des médias, est une région des États-Unis utilisée pour définir les marchés de la télévision et de la radio. Les droits de lecteur peuvent être utilisés pour limiter l'écoute en fonction des DMA.

  1. Entrez une liste des DMA dont l'affichage sera interdit (sur liste noire) ou autorisé (sur liste blanche)
  2. Cliquez sur Enregistrer.

Attribution de restrictions de domaine

Les restrictions de domaine vous permettent de limiter la lecteur des vidéos en fonction du domaine. Pour attribuer des restrictions de domaine à un droit de lecteur, procédez comme suit:

  1. Cliquez sur une lecteur à droite.
  2. Dans la navigation de gauche, cliquez sur Restrictions de domaine.
  3. Dans le champ Domaines sur liste noire, saisissez les domaines dont l'affichage ne sera pas possible.
  4. Dans le champ Domaines sur liste blanche, saisissez les domaines qui seront autorisés à être affichés.
  5. Cliquez sur Enregistrer.

Attribution de restrictions de proxy

Les restrictions de proxy vous permettent de limiter la lecteur vidéo en fonction de certains types de connexions. Pour attribuer des restrictions de proxy à un droit de lecteur, procédez comme suit:

Remarque: Alors que les fournisseurs de VPN continuent d'améliorer leurs services, la résolution des adresses IP peut changer par rapport à ce que vous avez défini pour les adresses IP bloquées. Pour cette raison, le blocage du proxy n'est pas garanti à 100 %.
  1. Cliquez sur une lecteur à droite.
  2. Dans le volet de navigation de gauche, cliquez sur Restrictions du
  3. Sélectionnez les types de proxy à bloquer.
  4. Cliquez sur Enregistrer.

Attribution de restrictions temporelles

Les restrictions temporelles vous permettent de limiter la lecteur des vidéos en fonction d'une date de début spécifiée. Pour assigner des restrictions temporelles à un droit de lecteur, procédez comme suit:

  1. Cliquez sur une lecteur à droite.
  2. Dans le volet de navigation de gauche, cliquez sur Restrictions de
  3. Cliquez sur le contrôle du calendrier pour sélectionner une date de début.
  4. (Facultatif) Cliquez sur Ajouter une date de fin pour sélectionner une date de fin.
  5. Cliquez sur Enregistrer.
