Les profils d'ingestion contiennent les paramètres de transcodage qui seront utilisés lorsque les vidéos seront téléchargées à l'aide du module Upload ou de l'API Dynamic Ingest.

Pour accéder à la page d'administration des profils d'ingestion, ouvrez le module Admin, puis Ingest Profiles. Notez que pour accéder à la page d'administration des profils d'ingestion, vous devez être un utilisateur {{site.product}} avec le rôle d'administrateur.

Une liste des profils d'acquisition s'affiche.

Remarque: Cliquez sur le lien Afficher le nom de l'API dans l'en-tête pour afficher le nom interne du profil d'ingestion. Le nom interne est utilisé lors de l'utilisation du nom du profil d'ingestion dans un appel d'API.

La page affichera les informations suivantes:

  • Nom et description du profil
  • Version du profil: indique le type de transcodage utilisé. Notez que les types qui apparaissent dépendent de la façon dont votre compte {{site.product_short}} est configuré:
    • Dynamic Delivery: le transcodage Dynamic Delivery est utilisé. Pour plus d'informations sur la livraison dynamique, consultez la section Présentation de Dynamic Delivery.
    • Context-Aware - Le transcodage contextuel est utilisé. Pour plus d'informations sur le codage sensible au contexte, reportez-vous à la section Présentation du codage sensible au contexte.
  • Type: type de profil d'ingestion
    • VOD standard: indique qu'il s'agit d'un profil par défaut fourni par {{site.product_short}}. Ce type de profil ne peut être ni modifié ni supprimé.
    • VOD personnalisé: indique qu'il s'agit d'un profil d'acquisition personnalisé.
    • Standard Live: indique qu'il s'agit d'un profil Live par défaut fourni par {{site.product_short}}. Ce type de profil ne peut être ni modifié ni supprimé.
    • Live personnalisé: indique qu'il s'agit d'un profil d'acquisition Live personnalisé.
  • Rendus: nombre de rendus vidéo pouvant être créés.
  • Afficher - Détermine si le profil est affiché et peut être sélectionné dans le module Upload.

Pour plus de détails sur les profils d'acquisition fournis par Brightcove (profils avec un type de VOD standard ou Standard Live), consultez l'un des documents suivants documents:

  • Dynamic Delivery ou Context-Aware Profile Version - Profils d'acquisition standard pour la diffusion dynamique et le codage sensible au contexte
  • Types Live personnalisés - Profils d'acquisition Live standard

Sur la page d'administration des profils d'ingestion, vous pouvez effectuer les opérations suivantes:

  • Créer de nouveaux profils d'ingestion
  • Modifier les profils d'ingestion
  • Supprimer les profils d'acquisition
  • Modifier le profil d'acquisition par défaut
  • Masquer/Afficher les profils d'ingestion

Création d'un nouveau profil d'ingestion

Remarque: La possibilité de créer et de modifier des profils d'ingestion n'est disponible que pour les éditeurs disposant de comptes Enterprise et Professional.

Pour créer un nouveau profil d'acquisition, cliquez sur Nouveau profil.

Le processus de création d'un nouveau profil d'acquisition varie en fonction du type de profil créé. Consultez l'un des documents suivants pour savoir comment créer un profil d'acquisition personnalisé:

  • Création de profils d'ingestion pour une diffusion dynamique
  • Création de profils d'ingestion pour le codage sensible au contexte
  • Création de profils Live Ingest personnalisés

Modification des profils d'ingestion

La page d'administration des profils d'ingestion peut être utilisée pour modifier les profils d'ingestion. Notez que l'ensemble de profils d'acquisition par défaut (marqué Standard VOD) ne peut pas être modifié.

Pour modifier un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur le lien Ingest Profiles.
  3. Cliquez sur le profil d'acquisition pour le modifier.
  4. Utilisez l'éditeur de profil pour modifier la définition du profil.
  5. Cliquez sur Enregistrer.

Suppression des profils d'acquisition

La page d'administration des profils d'ingestion peut être utilisée pour supprimer les profils d'acquisition qui ne sont plus nécessaires. Notez que l'ensemble de profils d'acquisition par défaut (marqué Standard VOD) ne peut pas être supprimé, mais ils peuvent être masqués s'ils ne sont pas nécessaires.

Pour supprimer un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur le lien Ingest Profiles. La page d'administration des profils d'ingestion s'ouvre et affiche les profils d'acquisition qui ont été créés pour votre compte.
  3. Cliquez sur l'icône de suppression () en regard du profil d'acquisition ou cliquez sur le nom du profil, puis sur Supprimer le profil.
  4. Confirmez la suppression en cliquant sur Supprimer le profil.

Sélection d'un profil d'acquisition par défaut

Un profil d'acquisition par défaut peut être configuré lorsque les vidéos sont téléchargées et lors de l'écrêtage à l'aide du module Social. Le profil par défaut de chargement est indiqué en regard du nom du profil. Le profil par défaut de l'extrait social est indiqué en regard du nom du profil.

Pour sélectionner un profil d'acquisition par défaut pour le téléchargement/découpage:

  1. Ouvrez le module Admin.
  2. Cliquez sur le lien Ingest Profiles. La page d'administration des profils d'ingestion s'ouvre et affiche les profils d'acquisition qui ont été créés pour le compte.
  3. Cliquez sur le nom du profil d'acquisition du profil que vous souhaitez définir comme profil par défaut.
  4. Cliquez sur le lien Uploading or Social Clipping en haut de la page pour définir le profil actuel comme profil par défaut.
Remarque: Lorsque vous sélectionnez un profil d'acquisition par défaut, vous devez vous assurer que le profil est visible et non masqué.
Remarque: Seuls les profils de diffusion dynamique peuvent être définis comme profil par défaut de découpage social. Les profils CAE ne sont pas pris en charge.

Masquer et afficher les profils d'ingestion

Les profils d'acquisition peuvent être masqués en cliquant sur le curseur associé au profil sur la position Masquer. Cela est utile pour masquer les profils par défaut qui ne sont peut-être pas nécessaires.

Les profils d'ingestion masqués n'apparaîtront pas dans la liste Profil d'acquisition du module Upload.
