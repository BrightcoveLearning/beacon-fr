Les paramètres de langue permettent de définir une langue principale, puis d'ajouter et de gérer des langues de métadonnées supplémentaires. La langue principale est la langue que votre compte utilise actuellement pour les métadonnées vidéo. Avant d'ajouter des langues supplémentaires à votre compte, vos métadonnées existantes seront associées à la langue que vous définissez comme langue principale. La langue principale est celle qui s'affiche initialement lorsque vous modifiez les propriétés de la vidéo dans le module Médias.

Définition d'une langue principale

Pour définir la langue principale:

  1. Connectez-vous à {{site.product}}.
  2. Ouvrez le module Admin, puis Paramètres linguistiques. Notez que vous devez être administrateur de compte pour accéder au menu Admin.
  3. Saisissez votre langue principale. Au fur et à mesure que vous tapez, une liste des langues correspondantes s'affiche.
  4. Cliquez sur une langue pour la sélectionner.
  5. Cliquez sur Enregistrer la langue.
Pour changer de langue principale, cliquez sur la langue, sélectionnez une nouvelle langue, puis cliquez sur Enregistrer.

Ajout de langues supplémentaires

Après l'enregistrement d'une langue principale, des langues de métadonnées supplémentaires peuvent être ajoutées. En ajoutant des langues supplémentaires, vous serez en mesure de saisir des métadonnées multilingues supplémentaires dans le module Médias. Pour ajouter d'autres langues, procédez comme suit.

  1. Cliquez sur Ajouter une langue.
  2. Saisissez votre langue. Au fur et à mesure que vous tapez, une liste des langues correspondantes s'affiche.
  3. Cliquez sur une langue pour la sélectionner.
  4. Cliquez sur Enregistrer la langue. La langue sera ajoutée au tableau Langues supplémentaires.
  5. Cliquez sur Ajouter une langue pour ajouter d'autres langues selon vos besoins.

Pour modifier le nom de la langue (tel qu'il apparaîtra dans le module Médias), cliquez dessus, apportez les modifications souhaitées, puis cliquez sur Enregistrer. Pour supprimer une langue, cliquez sur l'icône Supprimer ().

Remarque: Si vous supprimez une langue utilisée, toutes les métadonnées associées à cette langue seront masquées. Pour restaurer les métadonnées associées, ajoutez à nouveau la langue.
