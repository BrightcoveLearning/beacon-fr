L'outil de migration dirigée permet de spécifier qu'un lecteur Brightcove doit être chargé au lieu d'un lecteur intelligent. L'outil fournit un moyen simple de publier des vidéos dans les lecteurs Brightcove au lieu d'utiliser les anciens lecteurs intelligents. Aucune modification de votre code de publication existant n'est requise.

Cette rubrique contient les sections suivantes:

  • Exigences de migration dirigée
  • Création d'un mappage
  • Tester les mappages des lecteur s
  • Modification des mappages de lecteurs
  • Suppression des mappages de lecteur s
Remarque: Les mappages entre Smart Player et Brightcove Player continueront de fonctionner après la fin de vie de Smart Player fin 2017.

Exigences de la migration dirigée

Les exigences suivantes s'appliquent lors de l'utilisation de l'outil de migration dirigée:

  • L'interface utilisateur de migration dirigée ne s'affiche que lorsque vous avez 3 lecteurs intelligents ou plus sur votre compte
  • Vous devez créer un nouveau lecteur Brightcove doté du style, des personnalisations et des plug-ins compatibles avec le lecteur Brightcove
  • L'outil de migration dirigée n'est pas pris en charge pour les lecteur s disposant de listes de lecteur.
  • Tout code API Smart Player de votre page existante ne fonctionnera plus une fois le lecteur remplacé.

Limitations

Notez les limites suivantes de l'outil de migration dirigée:

  • La migration dirigée fonctionne lors de l'utilisation du code de publication intégré de Smart Player ou avec des URL raccourcies ou personnalisées (bcove.me ou link.brightcove.com).
  • La migration dirigée ne fonctionnera pas avec le code intégré Flash uniquement issu de Legacy Studio.

Création d'un mappage

Les mappages sont utilisés pour spécifier qu'un lecteur intelligent doit être remplacé par un lecteur Brightcove. Pour créer un nouveau mappage, procédez comme suit:

  1. Ouvrez le module Admin.
  2. Cliquez sur Paramètres du lecteur.
  3. Cliquez sur Create Mapping. Une liste des Smart Players du compte s'affiche.
  4. Sélectionnez un ou plusieurs lecteurs intelligents à remplacer par un lecteur Brightcove. Cliquez sur Suivant.
  5. Sélectionnez un lecteur Brightcove à utiliser à la place du ou des lecteurs intelligents. Lorsqu'un lecteur Brightcove est sélectionné, un aperçu du lecteur s'affiche.
  6. Cliquez sur Suivant. Les instructions pour tester les mappages des lecteur s seront affichées.

    Remarque: Les mappages de lecteur s peuvent prendre jusqu'à 5 minutes pour prendre effet.

  7. Cliquez sur Enregistrer. Une liste des mappages actuels s'affiche.

Après avoir créé un mappage, le statut sera Testing. Une fois le mappage testé et validé, le statut sera Committed. Les mappages dont le statut est Committed (Committed) afficheront un lecteur Brightcove au lieu du lecteur intelligent.

Tester les mappages des lecteur s

Vous devez tester tous les mappages de lecteur s créés avant qu'ils ne soient validés. Pour tester le mappage d'un lecteur, procédez comme suit:

  1. Trouvez une page qui affiche votre ancien lecteur Smart Player.
  2. Ajoutez le paramètre EnableMapping=True à l'URL de la page.
    • Si l'URL de la page ne comporte pas de paramètres d'URL, ajoutez ? EnableMapping=True à la fin de l'URL. Par exemple:
      http://www.mysite.com/videos.html?enableMapping=true
    • Si l'URL de la page contient des paramètres d'URL, ajoutez &EnableMapping=true à la fin de l'URL. Par exemple:
      http://www.mysite.com/videos.html?myparams=true&enableMapping=true

    Le lecteur Brightcove nouvellement mappé devrait s'afficher et la vidéo devrait être lue.

    Lorsque vous êtes satisfait du fonctionnement du lecteur, revenez sur {{site.product}}.

  3. Cochez la case en regard du mappage du lecteur et cliquez sur Commit Mapping pour rendre le nouveau lecteur Brightcove visible par tous les visiteurs du site.

Modification des mappages de lecteurs

Pour modifier le mappage d'un lecteur, procédez comme suit:

  1. Ouvrez le module Admin.
  2. Cliquez sur Paramètres du lecteur. Une liste des mappages de lecteur s s'affiche.
  3. Cochez la case en regard du mappage du lecteur à modifier. Cliquez sur Modifier le mappage.
  4. Sélectionnez un nouveau lecteur Brightcove vers lequel mapper. Cliquez sur Suivant, puis sur Enregistrer pour enregistrer les modifications.

Assurez-vous de tester toutes les modifications apportées aux mappages des lecteur s.

Suppression des mappages de lecteur s

Pour supprimer un mappage de lecteur, procédez comme suit:

  1. Ouvrez le module Admin.
  2. Cliquez sur Paramètres du lecteur. Une liste des mappages de lecteur s s'affiche.
  3. Cochez la case en regard du mappage du lecteur à supprimer. Cliquez sur Supprimer le mappage.
  4. Confirmez la suppression en cliquant sur Supprimer.
