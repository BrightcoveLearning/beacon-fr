Les utilisateurs de Studio peuvent utiliser la page Informations sur le profil pour afficher et modifier leurs propres informations de compte personnelles.

Pour accéder à la page Informations sur le profil:

  1. Connectez-vous à {{site.product}}.
  2. Dans la partie droite de l'en-tête de navigation, cliquez sur l'icône de profil, puis sur le lien Mon profil.

Modification des informations de profil

Tous les champs de la section Informations sur le profil peuvent être modifiés. Cliquez sur Enregistrer les modifications pour enregistrer les modifications.

Remarques sur le montage:

  • Vous ne pouvez pas modifier votre adresse e-mail. Si vous devez modifier votre adresse e-mail, vous devez créer un compte avec la bonne adresse. L'administrateur de compte principal peut le faire.
  • Vous pouvez modifier votre mot de passe, mais vous devez vérifier votre mot de passe actuel
  • Les mots de passe ne doivent pas contenir plus de 8 lettres et contenir au moins un chiffre et une lettre
  • Le fuseau horaire que vous définissez ici est utilisé lorsque vous planifiez la date et l'heure d'activation d'une vidéo. Vous pouvez planifier la date de début et de fin d'activation d'une vidéo en accédant au module Médias de Studio, en sélectionnant les détails d'une vidéo, puis en modifiant la section Disponibilité.
  • Vous pouvez utiliser la liste déroulante Langue pour sélectionner la langue de Studio:
    • Anglais
    • Espagnol (Espagne)
    • japonais
    • Français (France)
    • Chinois (traditionnel)
    • coréen
Remarque: Si vous êtes administrateur, vous pouvez utiliser la page Gestion des utilisateurs de Studio pour modifier le mot de passe et le rôle d'un utilisateur.
