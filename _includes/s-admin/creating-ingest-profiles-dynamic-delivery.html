Les profils d'acquisition contiennent les paramètres de transcodage qui seront utilisés lors du transcodage des vidéos. L'éditeur de profil d'acquisition qui s'affiche change en fonction des options de transcodage configurées pour le compte.

Remarques:
  • La possibilité de créer et de modifier des profils d'acquisition n'est disponible que pour les éditeurs disposant de comptes Enterprise et Professional.
  • Si vous souhaitez pouvoir retranscoder les vidéos ingérées avec un profil personnalisé, vous devez archiver le master numérique (comme le font tous les profils standard). Consultez Archiving Masters pour plus de détails sur la procédure à suivre.

Pour accéder à la page d'administration des profils d'ingestion, cliquez sur le lien Admin, puis sur Ingest Profiles. Une liste des profils d'acquisition s'affiche.

Remarque: Pour accéder à la page Profils d'acquisition, vous devez être un utilisateur {{site.product}} avec le rôle d'administrateur.

Votre compte Brightcove sera configuré avec un ou plusieurs profils d'acquisition. Tous les comptes Brightcove sont configurés avec un ensemble par défaut de profils d'acquisition qui ne peuvent être ni modifiés ni supprimés. Ceux-ci sont marqués d'un type de VOD standard dans le studio. La version du profil affiche le type de transcodage utilisé avec le profil. Pour plus de détails sur les rendus fournis avec les profils d'acquisition fournis par Brightcove, voir Profils d'acquisition standard pour la diffusion dynamique et l'encodage sensible au contexte.

Création d'un nouveau profil de diffusion dynamique

La page d'administration des profils d'ingestion peut être utilisée pour créer de nouveaux profils d'ingestion. Les profils d'ingestion sont définis à l'aide de JSON.

Pour créer un nouveau profil d'ingestion de diffusion dynamique:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Cliquez sur Nouveau profil > Diffusion dynamique. L'éditeur de profil s'ouvre.
  4. Entrez un nom et une description pour le profil.
  5. Décidez si vous souhaitez stocker les fichiers maîtres. Le stockage du fichier maître conserve une copie du fichier principal et permet de retranscoder la vidéo à l'avenir sans qu'il soit nécessaire de la télécharger à nouveau. Pour plus d'informations sur les fichiers maîtres d'archivage, consultez le document Masters d'archivage.
  6. Saisissez la largeur et la hauteur des images de l'affiche et des vignettes.
  7. Sélectionnez un paramètre pour les images:
    • Capturez les images automatiquement au moment du chargement: Brightcove capture les affiches et les vignettes lorsque la vidéo est téléchargée. Les images sont capturées à 50 % dans la vidéo.
    • Ne capturez pas d'images ; je téléchargerai ou capturerai mes propres images - Les affiches et les vignettes seront capturées ou téléchargées à l'aide du module Médias.
  8. Ajoutez les formats associés dont vous avez besoin pour le profil. Veillez à sélectionner les rendus vidéo et audio pour garantir la lecteur audio de vos vidéos. Ajoutez également des rendus progressifs (MP4) si vous souhaitez que des rendus téléchargeables soient disponibles pour la vidéo. Cliquez sur Ajouter un format associé.
    Remarques:
    • Pour afficher l'audio et les rendus disponibles, cliquez sur le lien Afficher la liste des formats associés.
    • Si vous souhaitez qu'un rendu 1080p soit disponible au téléchargement, vous devez créer un profil d'acquisition personnalisé et vous assurer qu'il inclut au moins l'un des formats associés suivants:
      • Progressif 3500
      • progressif 4000
    • Bien qu'il soit possible de créer un profil qui ne générera que des rendus progressifs, si vous chargez des vidéos à l'aide de ce profil, les images des vignettes et des affiches ne seront pas capturées lors du transcodage.
  9. Continuez à cliquer sur Ajouter un format associé pour ajouter des formats associés supplémentaires.
  10. Utilisez les listes déroulantes Codec et Rendition pour sélectionner un rendu audio, vidéo (H264, HEVC) ou progressif (MP4) à inclure. Les détails du format associé s'affichent dans le tableau.
  11. Cliquez sur Enregistrer pour enregistrer le nouveau profil.

Création de profils personnalisés pour Social Clipping

Les profils d'acquisition personnalisés peuvent également être utilisés pour le découpage social. En règle générale, les rendus de qualité inférieure inclus dans les profils d'acquisition par défaut de Brightcove ne sont pas nécessaires pour les cas d'utilisation de la distribution sociale. Par conséquent, les profils d'acquisition d'écrêtage personnalisés peuvent généralement être limités à un petit nombre de rendus de qualité moyenne à élevée. Par exemple, un profil d'acquisition d'écrêtage personnalisé qui inclut les rendus vidéo video3500, video2000 et video1700 convient parfaitement à la plupart des cas d'utilisation de la distribution sociale. L'utilisation d'un profil personnalisé pour le découpage social vous permet d'appliquer des transformations supplémentaires aux clips, telles que l'ajout d'images en filigrane. Pour créer un profil personnalisé avec un filigrane, contactez le support Brightcove.

Duplication des profils d'ingestion

Si vous devez créer un profil similaire à un profil existant, vous pouvez dupliquer un profil existant, puis le modifier pour apporter les modifications nécessaires.

Pour dupliquer un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Cliquez sur le profil d'acquisition pour le dupliquer.
  4. Cliquez sur Dupliquer
  5. Utilisez l'éditeur de profil pour modifier le nom et la définition du profil. Les formats associés peuvent être supprimés en cliquant sur l'icône de suppression () en regard d'un format associé.
  6. Cliquez sur Enregistrer.

Prise en charge DRM dans les profils

Contactez le support Brightcove pour définir les paramètres par défaut pour l'empaquetage DRM de vos vidéos. Il n'est pas nécessaire de spécifier un package DRM dans vos profils d'acquisition. Si votre compte est activé pour les DRM et que vous souhaitez diffuser certains titres sans DRM, modifiez les propriétés de la vidéo dans le module Médias et définissez l'indicateur DRM pour la vidéo sur Pas de DRM.

Remarque: Pour plus d'informations sur la création d'un lecteur Brightcove compatible DRM, consultez la rubrique Plug-in DRM.

Modification des profils d'ingestion

La page d'administration des profils d'ingestion peut être utilisée pour modifier les profils d'ingestion. Notez que l'ensemble de profils d'acquisition par défaut (marqué Standard VOD) ne peut pas être modifié.

Pour modifier un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Cliquez sur le profil d'acquisition pour le modifier.
  4. Utilisez l'éditeur de profil pour modifier la définition du profil. Les formats associés peuvent être supprimés en cliquant sur l'icône de suppression () en regard d'un format associé.
  5. Cliquez sur Enregistrer.
Remarque: vous pouvez ajouter des formats associés standard à un profil à l'aide de l'éditeur de base, mais des formats associés personnalisés doivent être ajoutés à l'aide de l'éditeur avancé.

Suppression des profils d'acquisition

La page d'administration des profils d'ingestion peut être utilisée pour supprimer les profils d'acquisition qui ne sont plus nécessaires. Notez que l'ensemble de profils d'acquisition par défaut (marqué Standard VOD) ne peut pas être supprimé, mais ils peuvent être masqués s'ils ne sont pas nécessaires.

Pour supprimer un profil d'acquisition:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles. La page d'administration des profils d'ingestion s'ouvre et affiche les profils d'acquisition qui ont été créés pour votre compte.
  3. Cliquez sur l'icône de suppression () en regard du profil d'acquisition ou cliquez sur le nom du profil, puis sur le bouton Supprimer.
  4. Confirmez la suppression en cliquant sur Supprimer le profil.

Sélection d'un profil d'acquisition par défaut

Un profil d'acquisition par défaut peut être configuré lorsque les vidéos sont téléchargées et lors de la création de clips à l'aide du module Social. Le profil par défaut de chargement est indiqué en regard du nom du profil. Le profil par défaut de l'extrait social est indiqué en regard du nom du profil.

Pour sélectionner un profil d'acquisition par défaut pour le téléchargement/découpage:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles. La page d'administration des profils d'ingestion s'ouvre et affiche les profils d'acquisition qui ont été créés pour le compte.
  3. Cliquez sur le nom du profil d'acquisition du profil que vous souhaitez définir comme profil par défaut.
  4. Cliquez sur le lien Uploading or Social Clipping en haut de la page pour définir le profil actuel comme profil par défaut.
Remarque: Seuls les profils de diffusion dynamique peuvent être définis comme profil par défaut de découpage social. Les profils CAE ne sont pas pris en charge.

Masquer et afficher les profils d'ingestion

Les profils d'acquisition peuvent être masqués en cliquant sur le curseur associé au profil sur la position Masquer. Cela est utile pour masquer les profils par défaut qui ne sont peut-être pas nécessaires.

Les profils d'ingestion masqués n'apparaîtront pas dans la liste Profil d'acquisition du module Upload.

Utilisation de l'éditeur avancé

Les profils d'ingestion sont définis à l'aide de JSON. Pour afficher la syntaxe JSON d'un profil, modifiez un profil, puis cliquez sur l'onglet Éditeur avancé. La représentation JSON du profil s'affiche.

Bien que le JSON puisse être modifié ici, Brightcove recommande de modifier les profils à l'aide de Simple Editor afin de réduire le risque d'erreurs. Pour plus d'informations sur les attributs disponibles, consultez la rubrique Ingest Profiles API Fields Reference for Dynamic Delivery and Context Aware Encoding.
