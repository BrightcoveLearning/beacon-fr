Création de profils Live Ingest personnalisés

Dans cette rubrique, vous apprendrez comment créer et gérer des profils d'acquisition Live personnalisés.

La page Administration des profils d'ingestion est utilisée pour gérer les profils d'ingestion Live personnalisés. Les profils d'acquisition Live déterminent le nombre et la qualité des rendus créés par Brightcove Live. Les profils d'acquisition Live ne sont pas utilisés lors du téléchargement de vidéos sur un compte Video Cloud. Les profils d'ingestion sont définis à l'aide de JSON. Avant de créer un nouveau profil d'acquisition, consultez le document Standard Live Ingest Profiles pour voir s'il existe un profil existant qui répondra à vos besoins.

Remarques:
  • Les comptes d'essai ne peuvent pas créer de profils
  • Chaque profil peut créer un maximum de 25 rendus
  • Vous pouvez créer des rendus pour la vidéo verticale. Il suffit d'inclure des valeurs de hauteur et de largeur dans les spécifications de rendu qui correspondent au rapport hauteur/largeur. de la vidéo (1280 pixels est la valeur maximale autorisée pour la hauteur)
  • Lorsque les profils en direct sont masqués dans Studio sous Admin > Ingest Profiles, ils ne peuvent pas être sélectionnés dans l'interface utilisateur Create Live Event, mais ils peuvent toujours utilisé avec l'API Live
  • Niveaux en direct:
    • Pour qu'un profil d'ingestion Live soit considéré comme Tier 1, la somme des débits binaires de tous ses rendus ne peut pas être supérieure à 5450
    • Si la somme des débits binaires de tous ses rendus est supérieure à 5 450 mais inférieure à 10 000, le profil d'ingestion Live sera considéré comme Tier 2.
    • Si la somme des débits binaires de tous ses rendus est supérieure à 10 000, le profil sera considéré comme Tier 3
Chaque profil peut créer un maximum de 25 rendus.

Création d'un nouveau profil

La page Administration des profils d'ingestion peut être utilisée pour créer des profils d'ingestion Live personnalisés. Notez que pour que le profil apparaisse dans l'interface utilisateur du module Live, il doit avoir model_version description: 1 dans le cadre des propriétés du profil.

Pour créer un nouveau profil d'acquisition Live personnalisé:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Cliquez sur Nouveau profil Ë‡ > En direct.
  4. Saisissez un nom pour le profil, puis cliquez sur Créer.
  5. Utilisez l'éditeur de profil pour modifier le profil selon vos besoins.
  6. Cliquez sur Enregistrer.

Duplication d'un profil

Il est possible de créer un nouveau profil en dupliquant un profil existant. Ensuite, vous pouvez modifier le nouveau profil avec vos modifications. Notez que pour que le profil apparaisse dans l'interface utilisateur du module Live, il doit avoir model_version description: 1 dans le cadre du profil.

Pour créer un profil d'acquisition Live personnalisé en dupliquant un profil existant:

  1. Ouvrez le module Admin.
  2. Cliquez sur Ingest Profiles.
  3. Localisez un profil avec un type de Standard Live à dupliquer. Cliquez sur le nom du profil pour ouvrir l'éditeur de profil.
  4. Cliquez sur Dupliquer L'éditeur de profil s'ouvre avec le nouveau profil.
  5. Faites défiler l'écran jusqu'en haut du profil. Modifiez l'élément name pour donner un nom significatif au profil. Ce nom sera affiché dans Video Cloud Studio.
  6. Modifiez l'élément de description pour donner au profil une description significative.
  7. Modifiez les informations de format associé si nécessaire.
  8. Cliquez sur Enregistrer.

Le profil nouvellement créé apparaîtra dans la liste des profils. Les profils personnalisés peuvent être supprimés en cliquant sur le bouton supprimer ().
