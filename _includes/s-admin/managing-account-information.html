La page Informations sur le compte gère le nom du compte et l'adresse postale et peut être utilisée pour cloner des comptes. Notez que pour accéder à la page Administration du compte, vous devez être un utilisateur ayant le rôle d'administrateur.

Pour accéder à la page Administration du compte:

  1. Connectez-vous à Studio.
  2. Ouvrez Admin, puis Informations sur le compte.

Les informations concernant le compte seront affichées.

Pour modifier les informations de votre compte, cliquez sur Modifier les informations du compte.

Modifiez les valeurs appropriées, puis cliquez sur Enregistrer. Notez que l'ID de compte ne peut pas être modifié.

Clonage de compte

Si votre compte est activé pour le clonage de compte, un bouton Créer un nouveau compte apparaît. La fonctionnalité de clonage de compte peut être utilisée pour créer de nouveaux comptes {{site.product_short}}. Lorsqu'un nouveau compte est créé, le support Brightcove en sera informé, puis travaillera avec votre responsable de compte pour vérifier le nouveau compte (sur la base de votre contrat). Une fois vérifié, le compte sera configuré avec les fonctionnalités décrites dans votre contrat. Une fois le compte entièrement configuré, vous en serez informé. Pendant la configuration, de nouveaux utilisateurs peuvent être ajoutés au compte.

Remarque: Le clonage de compte n'est disponible que pour les administrateurs de comptes disposant de comptes d'entreprise.

Pour cloner le compte, cliquez sur Créer un nouveau compte, entrez un nom de compte, puis cliquez sur Enregistrer.

Notez que le nouveau compte sera un compte d'essai lors de sa création. Une fois que le Support aura vérifié le statut du compte auprès de votre Account Manager, celui-ci sera automatiquement mis à jour pour indiquer Payé avec les options et services appropriés.
