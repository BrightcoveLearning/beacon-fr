Les lecteurs Brightcove configurés avec l'intégration de Google Analytics enverront des événements et des données personnalisées à un compte Google Analytics spécifique. Cette rubrique explique comment accéder aux données dans Google Analytics. Pour plus d'informations sur la façon d'activer l'intégration de Google Analytics pour un lecteur, consultez Configuration du suivi des événements Google Analytics.

Comment les données sont communiquées à Google Analytics

Les données Brightcove Player envoyées à Google Analytics apparaîtront dans l'interface Google Analytics avec les caractéristiques suivantes:

  • La catégorie d'événement est toujours Vidéo
  • L'action d'événement sera identique aux noms d'événements sélectionnés lors de la configuration de l'intégration. Ils incluent:
    • Impression vidéo
    • Demande de jeu
    • Engagement vidéo
    • Début de l'annonce
    • Fin de l'annonce
    • Charge du lecteur
    • Erreur
  • L'étiquette de l'événement sera au format Video ID Video Name
  • La valeur de l'événement sera vide pour tous les événements autres que Video Engagement. Pour Video Engagement, la valeur représente le pourcentage de visionnage

Affichage des données d'événements

Il existe de nombreuses manières d'afficher et de générer des rapports sur les données Google Analytics. Cette rubrique présente une façon de visualiser les données dans un format utile.

  1. Connectez-vous à votre compte Google Analytics.
  2. Utilisez le sélecteur en haut de la page pour sélectionner le bon compte et la propriété. Cela doit correspondre au compte sélectionné lors de la configuration de l'intégration dans le module Players.
  3. Sélectionnez une plage de dates à l'aide du sélecteur de dates en haut de la page.
  4. Dans le volet de navigation de gauche, cliquez sur Comportement > Événements > Présentation. Une liste des catégories d'événements s'affiche.
  5. Cliquez sur la catégorie d'événements vidéo. Les événements envoyés à Google Analytics par Brightcove Players auront la catégorie Vidéo comme catégorie d'événement.
  6. Pour la dimension principale, cliquez sur Action de l'événement.
  7. Tous les événements transmis par les lecteurs Brightcove seront répertoriés. Dans cet exemple, trois événements ont été passés.
  8. Cliquez sur une action d'événement pour afficher les détails. Dans cet exemple, cliquez sur Play Request pour afficher les vidéos avec une demande de lecteur. L'étiquette de l'événement contient l'ID et le nom de la vidéo.
