Étapes

  1. Ouvrez le module Analytics. Le rapport de performance s'affiche.
  2. Pour spécifier la période du rapport, utilisez le sélecteur de date en haut de la page.
  3. Sélectionnez cette option pour afficher la dimension Performance par vidéo.
  4. Cliquez sur la colonne Vues de vidéos pour trier les données.

Le champ de recherche en haut de la page peut être utilisé pour rechercher une vidéo spécifique.
