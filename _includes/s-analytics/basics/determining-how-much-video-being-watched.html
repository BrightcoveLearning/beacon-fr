Étapes

  1. Ouvrez le module Analytics.
  2. Cliquez sur Engagement dans la navigation de gauche.
  3. Pour spécifier la période du rapport, utilisez le sélecteur de date en haut de la page.
  4. Sélectionnez cette option pour afficher la dimension Engagement par vidéo.
  5. Cliquez sur la colonne Vues à 1 % pour trier les données.
  6. Cliquez sur le titre d'une vidéo pour afficher les données d'engagement de cette vidéo. Le graphique montre le nombre de visionnages de vidéos au fil du temps. Les pics dans le graphique indiquent qu'une partie de la vidéo a été regardée encore et encore. Les baisses soudaines de l'engagement doivent également être étudiées.
  7. Cliquez sur la ligne d'engagement pour voir cette partie de la vidéo dans le lecteur d'aperçu.

Le champ de recherche en haut de la page peut être utilisé pour rechercher une vidéo spécifique.
