Le module Analytics {{site.product_short}} fournit les rapports suivants:

  • Performance
  • Engagement
  • Tendance maintenant
  • Personnalisée

Un résumé de chacun de ces rapports est décrit dans les sections suivantes.

Rapport de performance

Le rapport de performance indique les performances des vidéos de votre compte, y compris les vues des vidéos, les impressions des vidéos, le taux de lecteur, les minutes visionnées et le score d'engagement. Le rapport de performance permet de segmenter les données en dimensions:

  • Vidéo: affiche les mesures de performance sur les vidéos
  • Player: affiche les mesures de performance des lecteur s
  • Source du trafic: affiche les mesures de performance basées sur la source du trafic
  • Domaine d'affichage: affiche les mesures de performance basées sur le domaine sur lequel se trouvait le spectateur lorsqu'une vidéo a été visionnée ; si le lecteur se trouve dans une iframe, le domaine d'affichage sera enregistré en tant qu'URL de la page dans l'iframe. (Cela correspond à la dimension destination_domain de l'API Analytics.)
  • Pays: affiche les statistiques de performance basées sur le pays dans lequel la vidéo a été visionnée
  • Appareil: affiche les mesures de performance pour chaque combinaison Device Type - Device Operating System qui a été utilisée pour visionner une vidéo
  • Type d'appareil: affiche les mesures de performance par type d'appareil
  • Système d'exploitation de l'appareil: affiche les mesures de performance par système d'exploitation
  • Device Manufacturer - Affiche les mesures de performance par fabricant de l'appareil
  • Type de navigateur: affiche les mesures de performance par type de navigateur
  • Day: affiche les mesures de performance en fonction du jour

Pour obtenir des informations détaillées sur le rapport de performance, consultez la référence du rapport de performance.

Rapport d'engagement

Le rapport d'engagement montre l'engagement pour les vidéos de votre compte. Cela inclut un score d'engagement et le nombre de vues aux points de 1 %, 25 %, 50 %, 75 % et 100 % de la vidéo. Le rapport d'engagement permet de segmenter les données en dimensions:

  • Vidéo: affiche les mesures d'engagement sur les vidéos
  • Player: affiche les mesures d'engagement des lecteur s
  • Day: affiche les mesures d'engagement par jour

Pour plus d'informations sur le rapport Engagement, consultez la référence du rapport d'engagement.

Rapport Trending Now

Le rapport Trending Now affichera un graphique montrant le nombre de spectateurs simultanés au cours de la dernière heure. Le rapport Trending Now peut être utilisé pour afficher les données de la VOD, des vidéos en direct ou des deux types de vidéos.

Pour obtenir des informations détaillées sur le rapport Tendance actuelle, reportez-vous au document de référence du rapport Tendance actuelle.

Rapports personnalisés

Le module Analytics peut être utilisé pour générer des rapports d'analyse ad hoc personnalisés qui peuvent être téléchargés au format de valeurs séparées par des virgules (.csv) ou Microsoft Excel (.xlsx).

Remarque: Les rapports personnalisés sont disponibles pour les éditeurs Enterprise et Pro.

Pour plus d'informations sur la création de rapports personnalisés, reportez-vous à la section Création de rapports Analytics personnalisés dans le module Analytics.
