Le module Analytics peut être utilisé pour générer des rapports d'analyse ad hoc personnalisés qui peuvent être exécutés une fois ou de manière récurrente. Pour plus d'informations sur la création de rapports personnalisés, consultez la section Création de rapports Analytics personnalisés. Tout rapport configuré pour être exécuté de manière récurrente apparaîtra dans la section Rapports planifiés du module Analytics.

Remarque: Les comptes d'entreprise peuvent avoir jusqu'à 20 rapports planifiés. Les comptes professionnels sont limités à 5 rapports.

Pour afficher la liste de tous les rapports planifiés, ouvrez le module Analytics, cliquez sur le lien Custom Reports dans la navigation latérale, puis sur le lien Scheduled Reports. Une liste des rapports planifiés s'affiche.

Afficher les détails du rapport

Pour afficher les détails du rapport, cliquez sur le signe + en regard du nom du rapport.

Édition de rapports

Pour modifier une définition de rapport, cliquez sur + en regard du nom du rapport pour ouvrir les détails du rapport, puis cliquez sur le lien Modifier. Apportez les modifications appropriées à la définition du rapport, puis cliquez sur Enregistrer le rapport.

Remarque: Si le rapport inclut plusieurs comptes, vous devez avoir accès à tous les comptes pour pouvoir le modifier.

Duplication de rapports

Pour dupliquer une définition de rapport, cliquez sur + en regard du nom du rapport pour ouvrir les détails du rapport, puis cliquez sur le lien Dupliquer. Une copie de la définition du rapport s'affiche. Apportez les modifications appropriées à la définition du rapport. Mettez à jour le nom du rapport, puis cliquez sur Planifier le rapport.

Remarque: Lors de la duplication d'un rapport, le type de rapport peut être modifié de Récurrent à Unique, ce qui vous permet d'exécuter le rapport immédiatement.

Suppression de rapports

Pour supprimer un rapport, cliquez sur + en regard du nom du rapport pour ouvrir les détails du rapport, puis cliquez sur le lien Supprimer. Confirmez la suppression en cliquant sur Supprimer le rapport.

Remarque: Si le rapport inclut plusieurs comptes, vous devez avoir accès à tous les comptes pour pouvoir le supprimer.

Modification du statut du rapport

Pour définir le statut d'un rapport, utilisez le curseur associé au rapport:

  • Actif - Le traitement du rapport est planifié
  • Inactif - Le rapport ne sera pas traité
