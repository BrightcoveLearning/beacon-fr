Affichage des données d'analyse pour le contenu Beacon

Dans Beacon, vous disposez de deux options pour afficher les données d'analyse, à savoir:

  • Google Analytics
  • Analyses du cloud vidéo

Google Analytics

Google Analytics recueille des données pour toutes les plateformes avec lesquelles vous utilisez Beacon. Vous pouvez ensuite utiliser les outils fournis par Brightcove pour afficher les données, comme indiqué dans le document Viewing Analytics Data.

Analyses du cloud vidéo

À l'heure actuelle, les plateformes suivantes prennent en charge Video Cloud Analytics:

  • Web
  • iOS
  • Apple TV
  • Android
  • TV Android

Pour plus d'informations sur l'affichage de Video Cloud Analytics, consultez les documents de la section Utilisation du module Analytics et des rapports Analytics de la page d'index Analytics. .
