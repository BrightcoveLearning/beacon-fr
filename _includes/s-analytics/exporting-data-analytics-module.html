Exportation des données de

Pour exporter les données d'un rapport, procédez comme suit.

  1. Ouvrez le module Analytics.
  2. Utilisez le sélecteur de date pour sélectionner une période.
  3. Sélectionnez un rapport.
  4. Cliquez sur le lien approprié pour exporter les données au format CSV ou XLSX.
  5. Enregistrez le fichier de données.
