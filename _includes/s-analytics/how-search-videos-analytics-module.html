Utilisation de la recherche rapide

La fonction Recherche rapide du module Analytics peut être utilisée pour rechercher une vidéo spécifique. Après avoir saisi les critères de recherche, la recherche rapide recherchera le nom de la vidéo, la description courte, la description longue, les balises, l'identifiant de la vidéo et tous les champs de texte personnalisés que vous pourriez avoir. Une liste des correspondances s'affichera sous le champ de recherche.

Notez que les vidéos supprimées ne sont pas consultables via la fonctionnalité de recherche, mais nous conservons des analyses pour elles. Vous pouvez les trouver en développant ou en parcourant le rapport de performance et en utilisant la fonction de recherche du navigateur ou en téléchargeant le rapport et en effectuant une recherche dans un tableur.

Cliquez sur le lien vers un match pour ouvrir le rapport qui est en contexte-fr/. Par exemple, si vous examinez l'engagement de votre compte, puis que vous utilisez la recherche rapide pour localiser une vidéo spécifique, cliquez sur le lien de la vidéo pour ouvrir le rapport d'engagement. Si vous consultez les rapports de performance, une recherche et un clic sur un lien ouvriront le rapport de performance.

Recherche de base

Si vous fournissez une valeur de recherche sans qualificatif, une recherche textuelle sera exécutée dans les champs suivants:

  • Nom
  • Brève description
  • Description longue
  • Balises
  • ID de la vidéo
  • ID de référence
  • Tous les champs personnalisés

Si vous souhaitez associer plusieurs mots, mettez le terme entre guillemets.

Recherche avancée

Le module Analytics permet également de rechercher des champs spécifiques, de rechercher en fonction de la date et d'inclure ou d'exclure des termes spécifiques.

Recherche de champs spécifiques

Pour rechercher une valeur spécifiée dans un champ spécifique, utilisez la syntaxe suivante:

  • champ:valeur à rechercher

Les champs suivants sont pris en charge:

terrain Valeurs supportées
nom chaînes ou chaînes entre guillemets
texte chaînes ou chaînes entre guillemets (valeurs dans le nom, description courte, description longue)
balises chaînes ou chaînes entre guillemets
reference_id chaîne ou chaînes entre guillemets
état ACTIF, INACTIF, SUPPRIMÉ, EN ATTENTE
updated_at voir Recherche de dates spécifiques pour le format de date
created_at voir Recherche de dates spécifiques pour le format de date
published_at voir Recherche de dates spécifiques pour le format de date
achevée vrai ou faux

Si vous voulez faire une correspondance exacte ou faire correspondre plusieurs mots, placez simplement le terme entre guillemets:

  • en direct-fr/
  • diffusion en direct-fr/

Cela fonctionne également lorsque vous effectuez une recherche dans un champ spécifique:

  • name: live-fr/
  • name: diffusion en direct-fr/

Recherche de dates spécifiques

Les champs de recherche dépendant de la date prennent des valeurs de plage de dates. Plusieurs formats de spécification de la date sont pris en charge:

Format Le sens
01/08/2013 Date au format aaaa-mm-jj
2013-08-01T 00:00:00 Z Une heure au format UTC
-1d Heure actuelle moins 1 jour (voir Dates relatives)
+1d Heure actuelle plus 1 jour (voir Dates relatives)
 

Dates relatives

Pour les dates relatives, nous prenons en charge une direction (+ ou -) suivie d'un nombre, suivie d'une durée (ou de son abréviation). Les durées légales sont les suivantes: minutes (m), heures (h), jours (d)

Par exemple:

  • - 1 jour
  • - 2 jours
  • + 4 heures
  • - 60 minutes
  • -1d
  • -2d
  • +4 h
  • -60 m

Plages de dates

Si le champ que vous recherchez est une date, vous pouvez spécifier une plage de dates à rechercher:

  • Updated_at:2013-08-01T 00:00:00 Z.. 2013-10-08T 00:00:00 Z

Cela recherchera toutes les vidéos avec une valeur updated_at entre le 1er août 2013 et le 8 octobre 2013.

Gammes ouvertes

Si vous souhaitez faire correspondre toutes les dates jusqu'à une heure donnée, ou faire correspondre toutes les dates depuis une heure donnée, omettez une extrémité de la plage. Par exemple, pour rechercher toutes les vidéos modifiées au cours des 2 derniers jours:

  • q=updated_at: -2 jours..

Rechercher toutes les vidéos modifiées avant le 11 août 2013:

  • q=updated_at:.. 2013-08-11T 00:00:00 Z

 

Y compris et excluant des conditions spécifiques

Vous pouvez marquer un terme comme obligatoire (les vidéos retournées DOIVENT correspondre) ou interdit (les vidéos renvoyées ne doivent PAS correspondre). Ceci est contrôlé par un signe + ou - immédiatement avant le terme:

Exemple Le sens
+animaux les vidéos DOIVENT inclure le terme animaux-fr/ dans le nom, la description courte ou la description longue
-animaux les vidéos ne doivent PAS inclure le terme animaux-fr/ dans le nom, la description courte ou la description longue
+nom: animals les vidéos DOIVENT inclure le terme animaux-fr/ dans le nom
-name: animals les vidéos ne doivent PAS inclure le terme animaux-fr/ dans le nom
