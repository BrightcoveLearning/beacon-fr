La plateforme d'analyse {{site.product_short}} est une plateforme d'analyse de pointe proche du temps réel, conçue à partir de zéro par Brightcove en utilisant les dernières innovations en matière de technologie Big Data et conçue pour une grande échelle. La plateforme d'analyse offre un accès rapide et quasiment en temps réel aux données d'analyse et inclut une puissante API ouverte qui permet aux développeurs de créer des solutions et des rapports d'analyse personnalisés en plus de ceux inclus dans {{site.product_short}} Studio.

Quels que soient vos objectifs spécifiques en matière de vidéo en ligne, comprendre votre public est essentiel pour développer une stratégie vidéo réussie. Mieux vous comprenez vos spectateurs, plus il est facile de créer du contenu qui les intéressera dans des formats qu'ils peuvent facilement consommer. Le module Analytics fournit différents rapports qui peuvent être utilisés pour obtenir des analyses sur vos vidéos et vos lecteurs:

  • Rapport de performance: fournit des détails sur le nombre de visionnages vidéo, les sources de trafic, les types d'appareils et les systèmes d'exploitation des appareils
  • Rapport d'engagement - Fournit des détails sur la manière dont les utilisateurs interagissent avec vos vidéos ; quel pourcentage de spectateurs regardent une vidéo
  • Tendance actuelle: fournit le nombre de visionneuses simultanées au cours de la dernière heure
  • Personnalisé: permet de créer des rapports d'analyse personnalisés

À l'aide de ces rapports, vous pouvez générer des rapports à jour sur le nombre de spectateurs que vous avez, la quantité de contenu que vos spectateurs regardent, le moment où ils arrêtent de regarder des vidéos et même les sources de trafic de votre contenu.

Notez que toutes les données affichées dans le module Analytics sont également disponibles via l'API Analytics. Pour plus d'informations, consultez la documentation de l'API Analytics.

Lancement du module Analytics

Pour lancer le module Analytics:

  1. Connectez-vous à {{site.product_short}} Studio.
  2. Ouvrez le module Analytics.

Sélection d'une période

Utilisez le sélecteur de date pour spécifier la période du rapport, vous pouvez sélectionner une plage de dates fixe ou sélectionner une plage de dates personnalisée. Notez que lorsqu'un seul jour est sélectionné, les graphiques s'affichent par heure.

Fuseau horaire de

Les données du module Analytics sont affichées en utilisant le fuseau horaire de reporting du compte. Le fuseau horaire s'affiche sous le contrôle du calendrier.

Le fuseau horaire de reporting est utilisé pour calculer les limites des jours pour les données de reporting. La modification du paramètre de fuseau horaire des rapports n'affecte que les données à venir, et les modifications ne sont pas appliquées rétroactivement. Si vous modifiez ce paramètre, vous pouvez voir un point plat ou un pic dans vos données lorsque vous examinez les limites des jours où la modification a été appliquée. Les mises à jour de ce paramètre peuvent ne pas prendre effet immédiatement et les données d'analyse peuvent continuer à faire référence à l'ancien paramètre jusqu'à ce que le système traite les modifications.

Le fuseau horaire des rapports peut être configuré dans le cadre des paramètres du compte. Pour plus d'informations sur la modification du fuseau horaire des rapports, consultez la section Gestion des paramètres du compte.

Recherche d'une vidéo

La fonction de recherche peut être utilisée pour rechercher une vidéo spécifique.

Les propriétés vidéo suivantes seront recherchées:

  • Nom
  • Brève description
  • Description longue
  • Balises
  • ID de la vidéo
  • ID de référence
  • Tous les champs personnalisés

Une liste des correspondances s'affichera sous le champ de recherche.

Cliquez sur un lien vidéo pour ouvrir le rapport en contexte-fr/. Par exemple, si vous examinez l'engagement de votre compte, puis que vous utilisez la fonction de recherche pour localiser une vidéo spécifique, cliquez sur le lien de la vidéo pour ouvrir le rapport d'engagement. Si vous consultez le rapport de performance, le fait de rechercher et de cliquer sur un lien permet d'ouvrir le rapport de performance.

Pour obtenir la liste complète des options disponibles lors de la recherche de vidéos dans le module Analytics, consultez Comment rechercher des vidéos dans le module Analytics.

Remarque: Les rapports Analytics incluent des analyses pour les vidéos supprimées de votre compte. Vous ne pouvez pas rechercher de vidéos supprimées dans le module Analytics. Pour obtenir un rapport sur une vidéo supprimée, vous devez connaître l'ID de la vidéo supprimée et utiliser l'API Analytics pour générer le rapport.
