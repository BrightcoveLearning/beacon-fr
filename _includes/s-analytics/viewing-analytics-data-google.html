Présentation

Il ne fait aucun doute que vous voudrez savoir ce que les spectateurs regardent dans vos applications Brightcove {{site.product_short}}. Pour ce faire, vous allez utiliser Google Analytics. Brightcove chargera un tableau de bord personnalisé et un rapport Google Data Studio personnalisé dans votre compte Google Analytics. Ces deux outils fournissent un excellent aperçu de la manière dont vos clients utilisent vos applications Brightcove {{site.product_short}}.

Outre le tableau de bord et le rapport personnalisés, vous pouvez utiliser les outils d'analyse standard de Google Analytics pour consulter le comportement des clients. Vous pouvez utiliser les données d'analyse standard collectées par Google. En outre, Brightcove ajoute d'autres entités à utiliser, détaillées plus loin dans ce document. Par exemple, les événements Movies sont distribués lorsqu'un film est visionné, une mesure Google Analytics par défaut, et le temps de visionnage est calculé, une mesure ajoutée par Brightcove. Par conséquent, dans Google Analytics, vous pouvez voir les films les plus regardés-fr/ en fonction du nombre de fois que les spectateurs ont lu le film et de la durée totale de visionnage. Ces entités ajoutées par Brightcove sont largement utilisées dans le tableau de bord personnalisé et le rapport mentionnés dans le paragraphe précédent.

Afficher le tableau de bord

Pour afficher le tableau de bord Google Analytics installé par Brightcove, et surtout les données qu'il affiche, procédez comme suit:

  1. Connectez-vous à Google Analytics.
  2. Cliquez sur le menu déroulant (surligné en jaune sur la capture d'écran) pour voir tous vos comptes Google Analytics.
  3. Dans la liste déroulante à trois panneaux, sélectionnez:

Visualisation des dimensions et des statistiques

Si vous n'êtes pas familier avec les dimensions et mesures d'analyse, voici une explication TRÈS simplifiée:
  • Les dimensions sont des attributs de vos données. Par exemple, la ville d'où proviennent les vues est une dimension. Une seule dimension peut contenir de nombreuses valeurs, comme les noms de toutes les villes dont vous avez des vues.
  • Les métriques sont des mesures quantitatives. Le nombre de sessions d'une ville donnée serait donc une mesure.

Brightcove ajoute des fonctionnalités aux mesures Google par défaut pour vos rapports d'analyse. Vous pouvez utiliser les dimensions et statistiques par défaut de Google Analytics, les éléments constitutifs des rapports d'analyse, ainsi que de nombreuses dimensions et une mesure ajoutées par Brightcove. Ces dimensions supplémentaires et la mesure unique sont utilisées dans le tableau de bord et le rapport Data Studio présentés dans les deux premières sections de ce document.

Notez que les dimensions et les statistiques ajoutées par Brightcove ne sont PAS les mêmes pour chaque type d'application Brightcove {{site.product_short}}. Par exemple, les données mobiles iOS ont 16 dimensions personnalisées ajoutées, tandis que l'application du site Web en a 20 ajoutées. Pour afficher la liste des dimensions et des statistiques dans l'interface utilisateur de Google Analytics, procédez comme suit:

  1. Connectez-vous à Google Analytics.
  2. Cliquez sur le menu déroulant (surligné en jaune sur la capture d'écran) pour voir tous vos comptes Google Analytics.
  3. Dans la liste déroulante à trois panneaux, sélectionnez:
    1. Votre compte Brightcove {{site.product_short}}
    2. Votre application pour laquelle vous souhaitez des analyses
    3. Votre point de vue (vous n'aurez qu'une seule vue par application)
  4. Dans le menu de navigation de gauche, cliquez sur l'option Admin.
  5. Cliquez sur Définitions personnalisées.
  6. Lorsque vous cliquez sur Définitions personnalisées, vous verrez des liens vers les dimensions personnalisées et les mesures personnalisées.
  7. Cliquez sur le lien Dimensions personnalisées pour voir les dimensions de cette application. Les dimensions changent selon l'application.
  8. Cliquez sur le lien Mesures personnalisées pour voir la mesure unique, Time Watched. Il s'agit de la seule mesure personnalisée pour les applications Brightcove {{site.product_short}}.

Informations sur le public

Vous pouvez consulter les statistiques de votre audience en suivant ces étapes:

  1. Choisissez votre application/vue en suivant les trois premières étapes de l'une des sections ci-dessus.
  2. Dans le menu de navigation de gauche, cliquez d'abord sur Audience, puis sur Aperçu.

  3. Sur le côté droit de l'écran, vous voyez un résumé de votre audience à l'aide de différentes mesures telles que la langue, les informations sur l'application et les appareils. Vous pouvez cliquer sur une mesure spécifique pour plus de détails.

Informations sur les écrans

Bien entendu, vous souhaiterez voir ce que font vos spectateurs dans vos applications. L'option pour cela est Behavior. Suivez ces étapes pour afficher les informations sur les affichages d'écran:

  1. Dans le menu de navigation de gauche, cliquez sur Comportement â†' Contenu du site â†' Toutes les pages.

  2. Sur le côté droit de l'écran, vous pouvez voir la liste des écrans les plus consultés.

Informations de lecteur et de visionnage

Toujours dans l'option Comportement, vous pouvez voir le nombre de fois où le bouton de lecteur d'une ressource vidéo a été cliqué, ainsi que la durée pendant laquelle la ressource a été réellement regardée. Suivez ces étapes pour obtenir ces informations:

  1. Tout en restant dans l'option Comportement, cliquez sur Événements â†' Principaux événements.
  2. Dans la liste des catégories d'événements, cliquez sur Lire des films
  3. Dans la liste Catégorie d'événement, cliquez sur Film.
  4. Dans la liste Action d'événement, cliquez sur Jouer.
  5. Notez que la liste indique le nombre de fois où le bouton de lecteur a été cliqué pour chaque élément.
  6. Notez que vous pouvez voir des informations sur une ou plusieurs de vos dimensions personnalisées en ajoutant la colonne appropriée au tableau. Pour ce faire, cliquez sur la flèche vers le bas de la liste déroulante Dimension secondaire, où vous pouvez voir le groupe Dimensions personnalisées.
  7. Vous pouvez maintenant ouvrir le groupe Dimensions personnalisées et sélectionner une dimension.
  8. Consultez le tableau d'exemple avec la dimension personnalisée Genre ajoutée.
  9. Notez que dans la liste Action de l'événement, vous pouvez également voir Time Watched, qui utilise bien entendu votre métrique personnalisée Time Watched.
