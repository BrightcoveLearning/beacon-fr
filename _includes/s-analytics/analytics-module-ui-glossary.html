Remarque: Pour obtenir la liste des éléments de données renvoyés par l'API Analytics et lors de l'exportation des données d'analyse, consultez le Glossaire de l'API Analytics.

UN

Médias actifs
Le nombre de vidéos actives sur le compte. Notez que ces données ne sont pas en temps réel, elles sont synchronisées à partir de votre vidéothèque une fois par jour. Notez également que ces données peuvent ne pas être exactes si vous consultez des données d'analyse en temps réel-fr/ (datant de moins de 3 jours).
API Analytics
Interface de programmation d'applications basée sur REST qui permet aux développeurs de récupérer les mesures d'analyse des vidéos Video Cloud directement au format JSON. L'API Analytics fournit les données de l'interface utilisateur Analytics dans Video Cloud Studio.

B

Bande passante totale
Bande passante totale utilisée pour diffuser des vidéos, des images et des charges de lecteurs. Notez que certaines parties des données de bande passante sont retardées, car elles proviennent de sources externes telles que des CDN. Normalement, les informations sont complètes dans les 48 heures, mais dans certains cas, la journée peut atteindre 5 jours.
Type de navigateur
Le type de navigateur. Les valeurs typiques sont les suivantes:
  • chrome
  • safari
  • firefox
  • c'est-à-dire
Octets livrés
Les données totales, en octets, fournies aux spectateurs, y compris les données vidéo/audio (y compris les données mises en mémoire tampon, qu'elles aient été lues ou non), les affiches et les vignettes, et toutes les données associées au lecteur vidéo.

C

ville
Ville où une vidéo a été visionnée.
Pays
Nom court en anglais du pays où la vidéo a été visionnée. Respecte la norme ISO 3166.
CSV
Valeurs séparées par des virgules. Les données Analytics peuvent être exportées dans ce format.

D

Utilisateurs uniques quotidiens
Il s'agit du nombre estimé de spectateurs uniques qu'une vidéo a eu un jour donné. Ces données ne sont disponibles que des données rapprochées (après 3 jours), et uniquement au niveau du compte. Pour plus d'informations, voir API Analytics: Utilisateurs uniques.
Journée
Le jour du mois où la vidéo a été visionnée.
Appareil
Combinaison du type d'appareil et du système d'exploitation de l'appareil utilisé pour visionner une vidéo.
Système d'exploitation du périphérique
Système d'exploitation de l'appareil utilisé sur l'appareil pour visionner une vidéo. Les valeurs possibles sont les suivantes:
  • Android
  • Bada
  • iOS
  • Linux
  • OS X
  • RIM (Blackberry)
  • Symbian
  • Système d'exploitation Web
  • Windows
  • Autre - pour les agents utilisateurs qui ne sont mappés à aucun des éléments ci-dessus
Fabricant d'appareils
Le fabricant de l'appareil. Voici quelques valeurs typiques:
  • lenovo
  • pomme
  • sony
  • microsoft
Type d'appareil
Type d'appareil utilisé pour visionner une vidéo. Les valeurs possibles sont les suivantes:
  • Ordinateur de bureau - comprend les ordinateurs de bureau et portables
  • Tablette - inclut les iPads, les tablettes Android, etc.
  • Mobile - téléphones portables
  • Téléviseur connecté/console de jeux/appareil compagnon - comprend les téléviseurs intelligents, les décodeurs et les consoles de jeux
  • Autre - pour les agents utilisateurs qui ne sont mappés à aucun des éléments ci-dessus
 
Dimension
Les dimensions sont des catégories de données analytiques sur lesquelles vous pouvez générer des rapports et des filtres. Les dimensions actuellement disponibles sont les suivantes:
  • Joueur
  • Vidéo
  • Pays
  • Journée
  • Afficher le domaine
  • Appareil
  • Type d'appareil
  • OS de l'appareil
  • Source du trafic
 
Afficher le domaine
Domaine référent signalé par le client (navigateur ou autre).
URL d'affichage
L'URL sur laquelle se trouvait le spectateur lorsqu'il a visionné une vidéo.

Pourcentage moyen regardé par vidéo vue (calculé en additionnant tout le pourcentage regardé divisé par le nombre de vues de vidéos). Pour plus d'informations sur le mode de calcul, consultez Calcul du score d'engagement vidéo.

Taux de jeu

Demande de jeu

T

Sources de trafic
Une indication de la manière dont le spectateur a accédé à la page du lecteur. Les valeurs possibles sont les suivantes:
  • Parrainage - L'utilisateur a suivi un lien vers la vidéo
  • Direct - L'utilisateur a tapé le lien directement, utilisé un signet ou utilisé un lien provenant d'un e-mail
  • Recherche organique - L'utilisateur a effectué une recherche et a cliqué sur un résultat pour accéder à la vidéo
  • Recherche payante - L'utilisateur a effectué une recherche et a cliqué sur un résultat payant pour accéder à la vidéo
  • Recherche sécurisée - L'utilisateur s'est connecté avec un compte Google, a effectué une recherche et a cliqué sur un résultat (payant ou organique ; les informations de recherche ne sont pas incluses dans ce cas)

Vidéo

Bande passante vidéo

Durée de la vidéo

Engagement vidéo
Mesure du pourcentage moyen regardé par vidéo visionnée (calculé en additionnant tout le pourcentage regardé divisé par le nombre de vues de vidéos). Pour plus d'informations sur le mode de calcul, consultez Calcul du score d'engagement vidéo.
Nom de la vidéo

Vues vidéo

Minutes consultées

Vues à 75 %

Vues à 100 %
Le nombre de vues enregistrées au point 100 % de la vidéo.
Impression vidéo
Le nombre de fois qu'une vidéo a été chargée dans un lecteur et prête pour l'interaction.
