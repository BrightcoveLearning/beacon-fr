Le module Analytics peut être utilisé pour générer des rapports d'analyse ad hoc personnalisés qui peuvent être téléchargés au format de valeurs séparées par des virgules (.csv) ou Microsoft Excel (.xlsx). Les rapports peuvent également être planifiés pour s'exécuter sur une base quotidienne, hebdomadaire ou mensuelle.

L'API Analytics est utilisée pour récupérer les données du système d'analyse. L'API Analytics peut également être utilisée pour créer vos propres rapports personnalisés. Pour plus d'informations sur l'API Analytics, consultez la documentation de l'API Analytics.

Pour commencer à créer des rapports personnalisés, ouvrez le module Analytics et cliquez sur le lien Personnalisé dans le menu de navigation de gauche.

Remarque: Vous remarquerez peut-être que le nombre de visionnages de vidéos pour une journée donnée varie au fil du temps. Pour savoir pourquoi cela peut se produire, consultez la FAQ sur Analytics.
Remarque: Les rapports renvoyant plus d'un million de lignes ne seront pas chargés.
Remarque: Les demandes de rapports personnalisés simultanées au sein d'un même compte seront exécutées en série. Si les rapports comportent de grandes plages de dates et que de nombreux champs sont sélectionnés, il peut y avoir un délai avant que tous les résultats ne soient renvoyés. Une bonne alternative consiste à planifier des rapports volumineux.
Remarque: la durée de la vidéo s'affiche en secondes

Pour créer un rapport personnalisé, vous allez effectuer les étapes suivantes:

  1. Sélectionnez les comptes
  2. Sélectionnez un type de rapport
  3. Sélectionnez des groupes de données
  4. Ajouter un filtre de rapport
  5. Sélectionnez les champs de données
  6. Afficher un aperçu du rapport
  7. Configuration des options de livraison
  8. Demandez le rapport

Fuseau horaire de

Les données des rapports personnalisés sont affichées en utilisant le décalage horaire du compte. Le fuseau horaire des rapports peut être configuré à partir de la page d'accueil de {{site.product_short}} Studio en accédant à Admin > Paramètres du compte > Fuseau horaire par défaut des rapports. Notez que seuls les administrateurs de compte ont accès aux paramètres du compte.

Le fuseau horaire de reporting est utilisé pour calculer les limites des jours pour les données de reporting. La modification du paramètre de fuseau horaire des rapports n'affecte que les données à venir, et les modifications ne sont pas appliquées rétroactivement. Si vous modifiez ce paramètre, vous pouvez voir un point plat ou un pic dans vos données lorsque vous examinez les limites des jours où la modification a été appliquée. Les mises à jour de ce paramètre peuvent ne pas prendre effet immédiatement et les données d'analyse peuvent continuer à faire référence à l'ancien paramètre jusqu'à ce que le système traite les modifications.

Notez que le fuseau horaire de reporting par défaut s'applique uniquement au module Analytics et non à Live Analytics, qui est techniquement un service séparé par sa conception.

Sélection des comptes

Si vous avez accès à plusieurs comptes Brightcove, vous pouvez inclure les données du compte courant ou sélectionner plusieurs comptes.

Si vous créez un rapport qui inclut tous les comptes et que ces comptes utilisent des fuseaux horaires différents, les données sont signalées en utilisant le fuseau horaire de ce compte.

Sélection d'un type de rapport

Sélectionnez le type de rapport:

  • Une seule fois - Le rapport sera exécuté une seule fois, immédiatement
  • Récurrent - Le rapport peut être programmé pour s'exécuter sur une base quotidienne, hebdomadaire ou mensuelle
    Remarque: Les rapports récurrents sont exécutés après le rapprochement des analyses, généralement 3 à 5 jours après la fin d'une période de reporting.

Rapport ponctuel

Si vous sélectionnez Une fois, le rapport sera exécuté une seule fois. Un contrôle de calendrier apparaît et vous pouvez spécifier la période que vous souhaitez que les données d'analyse couvrent.

Astuce: Commencez par sélectionner une plage de dates plus courte pour confirmer que le rapport contient les données que vous recherchez. Lorsque vous avez confirmé les données du rapport, étendez la plage de dates selon vos besoins.

Rapport récurrent

Si vous sélectionnez Récurrent, le rapport sera exécuté de manière récurrente. Sélectionnez un rapport Fréquence:

  • Quotidien - Le rapport couvre une journée entière
  • Hebdomadaire - Le rapport couvre une semaine entière, une semaine étant définie du lundi au dimanche
  • Mensuel - Les rapports couvrent un mois entier
  • Les comptes d'entreprise sont limités à 20 rapports récurrents, les comptes professionnels sont limités à 5
  • Les rapports récurrents sont associés à un utilisateur et sont disponibles sur tous les comptes auxquels l'utilisateur a accès
  • Les rapports récurrents sont exécutés après le rapprochement des analyses, généralement 3 à 5 jours après la fin d'une période de reporting

Sélection de groupes de données

Les groupes de données déterminent les champs de données qui peuvent être inclus dans le rapport. Les champs de données qui apparaissent dans la section Champs de données sont basés sur les groupes de données sélectionnés. Vous pouvez sélectionner plusieurs groupes de données, mais certaines combinaisons ne sont pas valides. Au fur et à mesure que les groupes de données sont sélectionnés, la liste des groupes de données disponibles sera mise à jour pour inclure uniquement les groupes valides pouvant être sélectionnés.

Remarques:
  • Le groupe de données Date ne peut être combiné à aucune autre dimension.
  • L'élément de données Octets livrés n'est plus disponible pour les rapports des lecteur s et sera grisé si vous sélectionnez des éléments de lecteur dans le groupe de données ou les données de performance colonnes.
  • Pour voir la liste des champs de données renvoyés avec chaque dimension, consultez ce document.

Groupes de données et champs

Lorsque vous sélectionnez un groupe de données, les champs de données pouvant être renvoyés peuvent être sélectionnés. Les champs grisés ne peuvent pas être renvoyés pour ce groupe de données.

Pour ajouter un groupe de données, cliquez sur le bouton bouton. Pour supprimer un groupe, cliquez sur .

Ajout d'un filtre de rapport

Des filtres peuvent être ajoutés pour limiter la quantité de données renvoyées. Lors de l'utilisation de plusieurs filtres, une opération AND sera utilisée. Dans l'exemple suivant, les vidéos marquées « video cloud essentials-fr/ ET dotées du champ personnalisé Product=Video Cloud seront renvoyées.

Remarque: Un filtre ne s'affichera dans votre rapport que si vous avez également sélectionné son groupe de données correspondant. Par exemple, si vous souhaitez appliquer un filtre sur la balise vidéo, vous devez sélectionner la dimension Vidéo.

Pour ajouter un filtre, cliquez sur le bouton bouton. Pour supprimer un filtre, cliquez sur . Les valeurs suivantes peuvent être utilisées comme filtres.

  • Vidéo: un ou plusieurs identifiants vidéo actifs dans le compte
  • Video Tag - Une ou plusieurs balises vidéo
    • Si vous saisissez services financiers-fr/ (entre guillemets), les vidéos étiquetées services financiers seront renvoyées.
    • Lorsque vous utilisez plusieurs balises, les vidéos comportant TOUTES les balises seront renvoyées. En entrant dans services financiers-fr/, la comptabilité renverra toutes les vidéos étiquetées services financiers et comptabilité.
    • Remarque: le mécanisme utilisé pour renvoyer les vidéos par tag ne comporte aucune option de tri ou de pagination.
  • Champs personnalisés: un ou plusieurs champs personnalisés du compte
    • Lorsque vous filtrez sur des champs personnalisés et que Sélectionner tout est sélectionné, la liste des champs personnalisés inclut les champs personnalisés de tous vos comptes. Au fur et à mesure que les champs personnalisés sont sélectionnés, la liste sera filtrée pour ne contenir que les champs personnalisés communs à tous les comptes.

Sélection des champs de données

La section Champs de données permet de sélectionner les valeurs de données qui seront incluses dans le rapport. La liste des champs de données disponibles est basée sur la ou les dimensions sélectionnées.

Si le compte sélectionné est configuré avec des champs personnalisés, ils peuvent également être sélectionnés. Si le rapport inclut tous les comptes, tous les champs personnalisés de ces comptes seront affichés.

Remarque: Pour les données d'utilisation, nous ne rapportons que les secondes diffusées en direct, et non la bande passante en direct.

Prévisualisation du rapport

La section Aperçu du rapport affiche les colonnes de données qui ont été sélectionnées, remplies avec des exemples de données. Cliquez sur l'icône d'une corbeille pour supprimer cet élément de données du rapport.

Remarque: Les données affichées dans l'aperçu du rapport sont des exemples de données, et non des données réelles de votre compte {{site.product_short}}. En outre, l'ordre de tri de l'aperçu n'a aucun effet sur la façon dont les données seront triées dans le rapport que vous téléchargez. Les données du rapport téléchargé peuvent être triées à l'aide d'un tableur tiers.

Configuration des options de livraison

Commencez par donner un nom significatif au rapport. Cela est particulièrement important lors de la création d'un rapport récurrent.

La section Format permet de sélectionner le format des données de rapport téléchargées.

  • CSV - valeurs séparées par des virgules
  • XLSX - Microsoft Excel

Les notifications par e-mail peuvent être utilisées pour vous informer ainsi que les autres utilisateurs de votre organisation lorsque le rapport est terminé. Cochez la case M'envoyer un e-mail lorsque mon rapport est prêt pour envoyer un e-mail au demandeur de rapport lorsque le rapport est terminé. D'autres personnes peuvent être informées en ajoutant leur adresse e-mail dans le champ de texte Notifier les autres. Une fois le rapport terminé, ils recevront un e-mail contenant un lien vers le rapport.

Le lien pour télécharger le rapport est valide pendant 90 jours.

Demander le rapport

Cliquez sur le bouton Demander un rapport (pour les rapports à usage unique) ou Planifier un rapport (pour les rapports récurrents) pour exécuter le rapport. S'il s'agit d'un rapport ponctuel, il sera téléchargé lorsqu'il sera terminé. Pour les rapports récurrents, le rapport sera mis en file d'attente et exécuté lorsque les données seront prêtes.

Affichage des rapports planifiés

Pour afficher tous les rapports récurrents, cliquez sur Rapports planifiés dans le volet de navigation de gauche. Consultez Gestion des rapports planifiés pour plus d'informations sur l'utilisation des rapports planifiés.
