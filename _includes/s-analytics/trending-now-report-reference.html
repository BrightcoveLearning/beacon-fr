Remarque description: Cette fonctionnalité est actuellement en version bêta. Si vous souhaitez l'activer pour votre compte, contactez votre responsable de compte.

Pour afficher le rapport Tendance actuelle, ouvrez le module Analytics et sélectionnez Tendance actuelle dans le volet de navigation de gauche. Le rapport affichera un graphique montrant le nombre de spectateurs simultanés au cours de la dernière heure. Le graphique sera mis à jour toutes les 45 secondes.

Remarque: Un utilisateur unique est déterminé par l'agent utilisateur du navigateur combiné à l'adresse IP. Plusieurs onglets d'un même lecteur s'exécutant dans le même navigateur ne représenteront qu'un seul utilisateur.

Le rapport Trending Now utilise le fenêtrage spéculatif pour calculer la valeur partiellement terminée. Ces agrégations émettent une nouvelle valeur minute toutes les 1 minute, puis une valeur complète à la fin de la minute. En fonction de la charge du système, le processus d'écrasement peut ne pas être terminé, ce qui peut entraîner des pics d'une minute dans les rapports de la CCU.

Si votre compte est activé pour Brightcove Live, vous aurez la possibilité de voir le nombre de spectateurs simultanés pour toutes les vidéos, ressources VOD ou ressources en direct.

Les sections sous le graphique afficheront les meilleures vidéos, les meilleurs pays et les meilleures villes de la dernière minute et seront mises à jour toutes les 5 secondes.
