La page Créer une chaîne en direct est l'endroit où vous allez créer une chaîne en direct. Les chaînes sont généralement utilisées pour diffuser des événements en direct 24h/24 et 7j/7

Pour créer un nouvel événement:

  1. Saisissez un nom de chaîne.
  2. Saisissez une brève description de l'événement (250 caractères maximum).
  3. Saisissez éventuellement des balises à associer à l'événement. Ceux-ci seront attribués à la vidéo si celle-ci est enregistrée en tant que ressource vidéo.
  4. Entrez un ID de référence comme identifiant facultatif pour un usage interne.
  5. Cliquez sur Create Channel.
