Accueil en direct

Le module Live est utilisé pour diffuser des événements en direct sur les ordinateurs de bureau et les appareils mobiles.

Créer un événement en direct

Cliquez sur Créer un événement, puis renseignez les informations suivantes:

  • Nom de l'événement
  • Brève description
  • En bas, cliquez sur le bouton Créer un événement pour démarrer l'événement en direct

Salle de contrôle
La Control Room contient toutes les informations et tous les paramètres de votre événement, dans cette fenêtre:
  • Copiez l'ID de la tâche et l'identifiant de la vidéo depuis le panneau Événement de test.

Vous aurez besoin des deux identifiants pour configurer l'événement en direct dans les paramètres de Brightcove Beacon Event.

Connectez votre Encoder
Pour démarrer le streaming sur la plateforme Live, vous aurez besoin d'un encodeur. Pour plus d'informations sur l'utilisation d'un encodeur, veuillez sélectionner:
  • Étape par étape: Utilisation du module Live avec Telestream Wirecast
  • Étape par étape: Utilisation du module Live avec Open Broadcaster Software (OBS)
