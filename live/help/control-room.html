La page Control Room est utilisée pour afficher/modifier les informations relatives à l'événement en direct.

Commencer le streaming

Pour connecter le flux vidéo à votre encodeur:

  1. Ouvrez votre encodeur pour démarrer la diffusion en continu.
  2. Cliquez sur Copier dans Streaming Endpoint (URL RTMP) et collez-le dans votre encodeur.
  3. Cliquez sur Copier dans le nom du flux et collez-le dans votre encodeur.
  4. Avec ces informations, lancez le streaming sur votre encodeur, et la vidéo apparaîtra dans la Control Room dans quelques secondes.
  5. Pour arrêter l'événement, cliquez sur Annuler le flux.

Informations sur les événements en

La page Control Room comporte cinq sections principales:

  • Informations sur l'événement: affiche des informations sur l'événement telles que Statut de l'événement, Visionneuses actuelles et Temps écoulé.
  • Détails de l'événement: affiche les détails de référence de l'événement tels que le nom de l'événement, l'ID de la vidéo et une brève description
  • Paramètres de configuration de l'événement: affiche des informations sur les paramètres de configuration de l'événement tels que l'ID de la tâche, l'ID vidéo, l'état et Endpoint.
  • Monitor - Récupère les données et les graphiques de l'événement en cours.
  • Formats associés: affiche des détails sur les formats associés qui ont été créés. Cliquez sur un lien de format associé pour afficher les paramètres d'encodage du rendu.
  • Clips: affiche tous les clips créés à partir du flux en direct.
