La page Control Room est utilisée pour afficher/modifier les informations relatives à la chaîne en direct.

Commencer le streaming

Pour connecter le flux vidéo à votre encodeur:

  1. Cliquez sur le bouton Activer. Les chaînes doivent être activées avant que le streaming puisse commencer.
  2. Ouvrez votre encodeur pour démarrer la diffusion en continu.
  3. Cliquez sur Copier dans Streaming Endpoint (URL RTMP) et collez-le dans votre encodeur.
  4. Cliquez sur Copier dans le nom du flux et collez-le dans votre encodeur.
  5. Avec ces informations, lancez le streaming sur votre encodeur, et la vidéo apparaîtra dans la Control Room dans quelques secondes.
  6. Pour arrêter la chaîne, cliquez sur Annuler le flux.

Informations sur les chaînes en

La page Control Room comporte cinq sections principales:

  • Informations sur la chaîne: affiche des informations sur l'événement telles que le statut de l'événement, les spectateurs actuels et le temps écoulé.
  • Détails de la chaîne: affiche les détails de référence de la chaîne tels que le nom de la chaîne, l'identifiant de la vidéo et une brève description.
  • Paramètres de configuration des canaux: affiche des informations sur les paramètres de configuration de l'événement tels que ID de tâche, ID vidéo, statut et Endpoint.
  • Monitor - Récupère les données et les graphiques de l'événement en cours.
  • Formats associés: affiche des détails sur les formats associés qui ont été créés. Cliquez sur un lien de format associé pour afficher les paramètres d'encodage du rendu.
  • Clips: affiche tous les clips créés à partir du flux en direct.
