Publié

Cette section affiche la liste des vidéos programmées et publiées à l'aide de Brightcove Social.

Avec la synchronisation automatique, vous pouvez voir plusieurs entrées pour la même vidéo:

  • Un pour la synchronisation initiale
  • Supplémentaire pour la synchronisation des métadonnées vidéo

Cliquez sur le lien du nom de la vidéo pour afficher les détails de la vidéo dans le module Médias.

Cliquez sur le lien du réseau social pour voir la page de la plateforme sociale.

Lister les données

Pour chaque vidéo publiée, vous verrez les données suivantes:

  • Statut
  • Heure de l'événement
  • Miniature de la vidéo
  • Nom de la vidéo et brève description
  • Lien vers le compte de réseau social et le site où la vidéo a été publiée
Publier des vidéos
  1. Cliquez sur Choisir les vidéos à publier
  2. Cela vous amène à la section Vidéos.
  3. À partir de là, vous pouvez sélectionner et publier des vidéos.
Filtrer par statut

Filtrez la liste par statut:

  • Tous les types d'état - Afficher tous les types d'état
  • Supprimé - Afficher les vidéos qui ont été publiées et supprimées d'une plateforme sociale
  • Error - Affiche les publications qui ont entraîné une erreur
  • Masqué - Afficher les publications dont la vidéo est privée, non répertoriée ou a expiré sur les réseaux sociaux
  • Traitement - Afficher les vidéos en cours de publication sur une plateforme sociale
  • Published - Afficher les vidéos qui ont été publiées avec succès sur une plateforme sociale
  • Programmé - Afficher les vidéos dont la publication est programmée sur une plateforme sociale
  • Synced - Afficher les vidéos synchronisées avec Auto Sync
  • Abandoned: affichez les vidéos qui ont été synchronisées automatiquement sur une plateforme sociale, puis
    • Supprimé de Video Cloud sans suppression sur la plateforme sociale ; Auto Sync a été configuré pour ne rien faire lorsqu'une vidéo est supprimée
    • Non balisé afin qu'il ne corresponde plus à la synchronisation automatique sans suppression sur la plateforme sociale ; Auto Sync a été configuré pour ne prendre aucune action lorsque les métadonnées vidéo changent™
Filtrer par destination

Filtrez la liste par destination de la plateforme sociale:

  • Toutes les destinations - Afficher les vidéos publiées sur toutes les plateformes sociales
  • Facebook - Afficher les vidéos publiées sur Facebook
  • Twitter - Afficher les vidéos publiées sur Twitter
  • YouTube - Afficher les vidéos publiées sur YouTube
Rechercher des vidéos

Utilisez le champ de recherche pour trouver des vidéos dans votre compte ; les champs suivants sont recherchés:

  • ID de la vidéo
  • Nom
  • Brève description
  • Description longue
  • Balises
  • ID de référence
  • Champs personnalisés
