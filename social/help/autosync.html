Synchronisation automatique

Synchronisez automatiquement les vidéos sur les réseaux sociaux en fonction des balises associées à une vidéo.

Avant de pouvoir publier des vidéos, vous devez créer des destinations pour vos comptes sociaux. Dans le volet de navigation de gauche, cliquez sur Destinations.

La publication manuelle de vidéos est indépendante de la synchronisation automatique. Si une vidéo est marquée pour être publiée avec Auto Sync, elle le fera même si vous l'avez publiée manuellement.
Configuration de la synchronisation automatique
  1. Cliquez sur Configurer la synchronisation automatique.
  2. Entrez un nom pour la synchronisation.
  3. Sélectionnez les vidéos à distribuer
    • Vidéos avec tous ces tags - Les vidéos avec TOUS les tags listés doivent être synchronisées
    • Vidéos avec l'un de ces tags - Les vidéos avec N'IMPORTE LAQUELLE des balises répertoriées doivent être synchronisées
    • Toutes les vidéos de ce compte: toutes les vidéos du compte doivent être synchronisées
  4. À moins que l'option Toutes les vidéos de ce compte n'ait été sélectionnée, entrez une liste de balises séparées par des virgules.
  5. Exclure les vidéos qui sont
    • Inactive - Les vidéos inactives ne seront pas synchronisées (par défaut)
    • Horaire restreint - Les vidéos programmées ne seront pas synchronisées
    • Géographiquement restreint - Les vidéos soumises à des restrictions géographiques ne seront pas synchronisées
  6. Cliquez sur Suivant.
  7. Choisissez une destination de publication pour synchroniser les vidéos.
  8. Pour créer de nouvelles destinations ou modifier des destinations existantes, cliquez sur Gérer les destinations.
  9. Cliquez sur une destination pour afficher les valeurs par défaut de la destination.
  10. Cliquez sur Suivant.
Définissez les règles YouTube

Des règles peuvent être définies pour gérer ce qu'il faut faire lorsqu'une vidéo est supprimée de Video Cloud, lorsque les balises changent ou lorsque les métadonnées d'une vidéo changent.

Les règles sont basées sur la plateforme de médias sociaux.

  1. Sélectionnez ce qui se passe lorsqu'une vidéo n'est pas balisée ou est supprimée de Video Cloud.
    • Supprimer la vidéo distribuée sur YouTube - La vidéo sera supprimée de YouTube
    • Rendre la vidéo distribuée - Rendre la vidéo YouTube privée ou non répertoriée
    • Aucune action - Rien ne se passe avec la vidéo YouTube
  2. Sélectionnez ce qui se passe lorsque les métadonnées de la vidéo sont modifiées.
    • Mettre à jour les métadonnées vidéo sur YouTube: les modifications apportées aux métadonnées vidéo dans Video Cloud seront synchronisées avec YouTube
    • Aucune action - Aucune action n'est entreprise
  3. Cliquez sur Suivant.
  4. Confirmez les paramètres de synchronisation automatique et cliquez sur Terminer.

Afficher/modifier les tâches de synchronisation automatique

Vous pouvez afficher et modifier les tâches de synchronisation automatique que vous avez créées.

  • Nom et ID de synchronisation automatique: cliquez sur un lien dans cette colonne pour ouvrir les propriétés de synchronisation
  • Tags - Cliquez sur un lien dans cette colonne pour ouvrir le module Médias avec une liste de vidéos portant le tag
  • Destination: cliquez sur un lien dans cette colonne pour afficher les détails de la destination
  • Delete: supprime la tâche de synchronisation associée

Propriétés de synchronisation

  • Statut
    • ACTIVE - Les vidéos seront synchronisées
    • PAUSED - Aucune vidéo ne sera synchronisée
  • Afficher - Filtrer la liste d'affichage
  • Paramètres: modifiez les paramètres de synchronisation ; cliquez sur Mettre à jour pour enregistrer les modifications
  • Delete - Supprimer la tâche de synchronisation
