Achats

Une liste des achats intégrés effectués par l'utilisateur.

Détails de l'achat

  • Date: date à laquelle l'utilisateur a effectué un achat intégré
  • Description du produit - Description de l'article acheté
  • Type: type d'achat
  • Store Name: nom de la boutique en ligne où le produit a été acheté
Supprimer et oublier

Cette action est utilisée pour la conformité au RGPD, lorsqu'un utilisateur demande que ses données personnelles soient effacées.

AVERTISSEMENT: La suppression d'une visionneuse entraîne la suppression de toutes les données et autorisations associées. Cela ne peut pas être annulé !
  1. Dans le volet de navigation de gauche, cliquez sur Supprimer et oublier.
  2. Dans la boîte de dialogue Supprimer et oublier, saisissez le mot Supprimer.
  3. Cliquez sur Supprimer et oubliez.
