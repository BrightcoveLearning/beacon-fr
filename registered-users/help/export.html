Exporter

Exportez les données des utilisateurs enregistrés sous forme de fichier CSV.

Exporter les données utilisateur

Sélectionnez l'une des options, et vous serez invité à filtrer les données que vous souhaitez inclure.

  • Tous les utilisateurs enregistrés - Exportez les données de tous les utilisateurs enregistrés de votre compte
  • Nouveaux utilisateurs - Exportez les données des nouveaux utilisateurs enregistrés pour le mois dernier
