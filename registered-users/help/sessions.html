Sessions actives

Voici les sessions actives associées à un utilisateur enregistré.

Détails de la session

  • Date: date à laquelle une session a été démarrée ; cliquez sur l'en-tête pour contrôler l'ordre de tri
  • Device: type d'appareil utilisé pour une session
  • Start time: heure à laquelle une session a été démarrée
  • Fin de session - Cliquez sur associé à une session pour la supprimer ; cela libérera une licence et vous permettra de vous connecter à une session disponible
  • Arrêter toutes les sessions: cliquez sur Terminer toutes les sessions pour mettre fin à toutes les sessions actives pour cet utilisateur
Supprimer et oublier

Cette action est utilisée pour la conformité au RGPD, lorsqu'un utilisateur demande que ses données personnelles soient effacées.

AVERTISSEMENT: La suppression d'une visionneuse entraîne la suppression de toutes les données et autorisations associées. Cela ne peut pas être annulé !
  1. Dans le volet de navigation de gauche, cliquez sur Supprimer et oublier.
  2. Dans la boîte de dialogue Supprimer et oublier, saisissez le mot Supprimer.
  3. Cliquez sur Supprimer et oubliez.
