Aperçu de l'utilisateur

La vue d'ensemble montre certains détails pour l'utilisateur enregistré sélectionné.

Informations sur l'utilisateur

  • E-mail: adresse e-mail de l'utilisateur
  • Name: nom complet de l'utilisateur
  • Dernière connexion: date à laquelle cet utilisateur s'est connecté pour la dernière fois à votre application
  • Abonnements actifs: liste des plans d'abonnement définis pour cet utilisateur ; cliquez sur le nom d'un plan pour voir les détails
  • Contrôle parental: indique si le contrôle parental est activé pour cet utilisateur
  • Dernière transaction: nom et date de la dernière transaction utilisateur ; cliquez sur le nom de la transaction pour voir les détails
Supprimer et oublier

Cette action est utilisée pour la conformité au RGPD, lorsqu'un utilisateur demande que ses données personnelles soient effacées.

AVERTISSEMENT: La suppression d'une visionneuse entraîne la suppression de toutes les données et autorisations associées. Cela ne peut pas être annulé !
  1. Dans le volet de navigation de gauche, cliquez sur Supprimer et oublier.
  2. Dans la boîte de dialogue Supprimer et oublier, saisissez le mot Supprimer.
  3. Cliquez sur Supprimer et oubliez.
