Présentation

Un utilisateur enregistré est une personne qui a créé un compte avec votre application Brightcove Beacon. Il peut s'agir d'un compte gratuit ou d'un abonnement payant.

Lorsqu'un utilisateur enregistré a une question ou un problème avec son compte, vous pouvez le rechercher rapidement à l'aide d'une recherche intelligente et vérifier ses informations. La première chose que vous verrez est une liste des utilisateurs enregistrés.

Cliquez sur un lien E-mail dans la liste pour afficher plus d'informations sur l'utilisateur. Pour chaque utilisateur, vous disposerez des sections d'onglets suivantes:

  • Aperçu
  • Afficher l'historique
  • Abonnements
  • Sessions actives
  • Profils
  • Supprimer et oublier

Vous pouvez également exporter des données détaillées sur vos utilisateurs enregistrés. Ces données peuvent être utilisées pour vous aider à gérer les campagnes de marketing par e-mail et les programmes de relation client.

  • Exporter les données des utilisateurs enregistrés

Liste des utilisateurs

Dans Beacon Studio, dans la navigation principale, cliquez sur Utilisateurs enregistrés pour afficher la liste des utilisateurs de votre compte.

La liste des utilisateurs ne contiendra que les utilisateurs que vous avez recherchés et que vous avez consultés. Si aucun utilisateur ne figure dans la liste, commencez à saisir son adresse e-mail ou son nom dans le champ Rechercher des utilisateurs.

Actions

Pour les détails de l'utilisateur, procédez comme suit:

  • Cliquez sur un lien E-mail dans la liste des utilisateurs
  • Dans la zone de recherche, commencez à saisir une adresse e-mail ou un nom d'utilisateur ; sélectionnez un utilisateur dans la liste des résultats

Des informations détaillées sur un utilisateur spécifique sont décrites dans les sections suivantes de ce document.

Aperçu

L'onglet Aperçu affiche des informations générales sur l'utilisateur sélectionné, y compris les noms des plans auxquels il s'est inscrit et ses paramètres de contrôle parental.

Champs

  • E-mail: adresse e-mail de cet utilisateur
  • Nom: prénom et nom de famille de l'utilisateur
  • Dernière connexion: date à laquelle l'utilisateur s'est connecté pour la dernière fois à votre application
  • Role: rôle assigné à l'utilisateur, si aucun N/A n'est affiché
  • Abonnements actifs - Une liste des plans d'abonnement auxquels l'utilisateur est inscrit
    Notez que si un package AuthVOD est activé, ce champ affichera N/A , comme suit:
  • Contrôle parental: indique si l'utilisateur a activé le contrôle parental pour son compte et si un code PIN est requis pour y accéder
  • Dernière transaction: date à laquelle l'utilisateur a effectué une transaction de monétisation pour la dernière fois

Actions

  • Cliquez sur le nom d'un abonnement actif pour voir les détails

Détails du plan

Champs

  • Nom: nom du plan d'abonnement
  • Description - Résumé du plan d'abonnement
  • Status: statut actuel du plan (qu'il soit publié ou non) et statut de l'abonnement de l'utilisateur
  • Options - Options de plan supplémentaires disponibles pour l'utilisateur

Afficher l'historique

L'historique de visionnage affiche la liste des films que cet utilisateur enregistré a regardés. L'histoire est capturée et sauvegardée pendant 1 an.

Colonnes

  • Date: date à laquelle un film a été visionné ; cliquez sur cet en-tête de colonne pour contrôler l'ordre de tri des résultats (décroissant ou croissant)
  • Titre: nom du film visionné
  • Profile: nom du profil utilisateur

Actions

  • Utilisez la liste déroulante Afficher le filtre pour limiter les résultats d'affichage
  • Cliquez sur le nom d'un titre pour voir les détails du film

Détails du film

Champs

  • Titre: nom du film, y compris la saison et l'épisode
  • Genre - Catégorie descriptive du film
  • Sortie - Année de sortie du film
  • Duration: durée du film en heures et minutes
  • Disponible - Disponibilité des films
  • Plans: liste des plans auxquels l'utilisateur est abonné
  • Asset ID: identifiant unique pour le film

Actions

  • Cliquez sur le numéro d'identification de l'actif pour afficher plus de détails dans le module Médias

Abonnements

L'onglet Abonnements affiche la liste des achats et des modifications de plans d'abonnement pour un utilisateur enregistré. Cela inclut les périodes d'essai du plan.

Colonnes

  • Nom du plan: nom du plan d'abonnement
  • Nom de la boutique: nom de l'App Store
  • Status: statut actuel de l'abonnement ; valeurs:
    • En période d'essai: l'utilisateur est en période d'essai pour ce plan
    • En attente - Les informations de paiement de l'utilisateur sont en cours
    • Actif - L'utilisateur s'est inscrit à ce plan d'abonnement
  • Start: date de début de l'abonnement associé
  • Fin de l'essai: date de fin de la période d'essai de l'utilisateur
  • Ends: date de fin de l'abonnement associé
  • Cancelled: date à laquelle l'abonnement a été annulé

Actions

  • Utilisez la liste déroulante Afficher le filtre pour limiter les résultats d'affichage
  • Cliquez sur le nom du plan pour afficher plus de détails sur le plan

Détails du plan

Champs

  • Nom: nom du plan d'abonnement
  • Description - Résumé du plan d'abonnement
  • Status: statut actuel du plan (qu'il soit publié ou non) et statut de l'abonnement de l'utilisateur
  • Options - Options de plan supplémentaires disponibles pour l'utilisateur

Ajouter un abonnement

Vous pouvez sélectionner un nom de plan ou, comme indiqué ici, vous pouvez ajouter manuellement un abonnement:

  1. Cliquez sur Ajouter un abonnement.

  2. Sélectionnez des valeurs pour chacun des champs de la boîte de dialogue. Tous les champs sont obligatoires.

    • Nom du plan - Sélectionnez un nom de plan d'abonnement que vous avez défini dans votre compte
    • Début - Entrez la date de début de l'abonnement
    • Fin - Entrez la date de fin de l'abonnement
  3. Cliquez sur Ajouter un abonnement.

Achats

L'onglet Achats affiche la liste des achats intégrés pour un utilisateur enregistré.

Colonnes

  • Date: date à laquelle l'utilisateur a effectué un achat intégré
  • Description du produit - Description de l'article acheté par l'utilisateur
  • Type: type d'achat
  • Store Name: nom de la boutique en ligne où le produit a été acheté

Sessions actives

Afficher la liste des sessions actives associées à un utilisateur enregistré.

Colonnes

  • Date: date à laquelle une session a été démarrée ; cliquez sur cet en-tête pour contrôler l'ordre de tri des résultats (décroissant ou croissant)
  • Device: type d'appareil utilisé pour une session
  • Start time: heure à laquelle une session a été démarrée
  • Fin de session - Cliquez sur associé à une session pour la supprimer

Actions

  • Cliquez sur Terminer toutes les sessions pour mettre fin à toutes les sessions utilisateur actuellement actives

Profils

Afficher la liste des profils d'utilisateurs associés à un utilisateur enregistré.

Colonnes

  • Profile: nom du profil
  • Created: date et heure de création

  • Supprimer le profil - Cliquez sur associé à un profil pour le supprimer

    Remarques:
    • Le profil principal ne peut pas être supprimé
    • L'activité et les données liées à ce profil ne seront plus suivies

Supprimer et oublier

Cette action est utilisée pour la conformité au RGPD, lorsqu'un utilisateur demande que ses données personnelles soient effacées.

AVERTISSEMENT: La suppression d'une visionneuse entraîne la suppression de toutes les données et autorisations associées. Cela ne peut pas être annulé !

Étapes

  1. Cliquez sur Supprimer et oubliez.

  2. Pour continuer, tapez Supprimer dans la zone de texte.

  3. Pour confirmer, cliquez sur Supprimer et oublier.
