Présentation

Un utilisateur enregistré est une personne qui a créé un compte avec votre application Brightcove Beacon. Il peut s'agir d'un compte gratuit ou d'un abonnement payant.

Vous pouvez exporter des données détaillées sur vos utilisateurs enregistrés. Ces données peuvent être utilisées pour vous aider à gérer les campagnes de marketing par e-mail et les programmes de relation client.

Exporter

Pour votre commodité, vous pouvez exporter des données détaillées sur vos utilisateurs enregistrés sous forme de fichier CSV. Vous pouvez ensuite ouvrir ce fichier dans un tableur pour gérer les campagnes de marketing par e-mail et les programmes de relation client.

Étapes

  1. Dans Video Cloud Studio, accédez au module Utilisateurs enregistrés.

  2. Cliquez sur Exporter.

  3. Sélectionnez Exporter pour l'un des éléments suivants: (Vous serez invité à filtrer les données que vous souhaitez inclure)

    • Tous les utilisateurs enregistrés - Exporter les données de tous les utilisateurs enregistrés
    • Nouveaux utilisateurs - Exportez les données des nouveaux utilisateurs enregistrés au cours du dernier mois

  4. Choisissez les données que vous souhaitez inclure dans votre fichier d'exportation.

    Plage de données

    • Plage de données: sélectionnez une période pour les données exportées

    Les données

    • E-mail: adresse e-mail de l'utilisateur
    • Nom: nom d'utilisateur
    • ID externe de l'utilisateur: ID utilisateur unique qui connecte l'utilisateur de l'application authentifié OIDC à un utilisateur de l'application Beacon
    • Jeton utilisateur: ID utilisateur unique généré par le système pour spécifier l'utilisateur de l'application Beacon
    • Date d'adhésion: date à laquelle l'utilisateur s'est inscrit à votre application
    • Dernière connexion: date de la dernière connexion de l'utilisateur à votre application
    • Détails de l'abonnement - Inclut le nom de l'abonnement, les dates de début/fin, le statut, la passerelle de paiement et l'ID

  5. Cliquez sur Exporter les données.
